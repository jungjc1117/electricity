     2.
     3.
     4.

해설

📘 문제 해설 요약 및 설명

이 문제는 전기 회로를 블록선도로 표현하는 문제입니다. 주어진 식:

E(s)=Z(s)I(s)=(R+sL)I(s)E(s) = Z(s)I(s) = (R + sL)I(s)

는 전기회로의 임피던스를 주파수 영역에서 나타낸 것으로, 다음과 같은 내용을 담고 있습니다:

🔧 해석

  • E(s): 입력 전압의 라플라스 변환 표현

  • I(s): 입력 전류의 라플라스 변환 표현

  • Z(s): 임피던스, 여기서는 $$Z(s) = R + sL$$로 표현됨

  • 이 식은 오옴의 법칙 $$E = ZI$$를 주파수 영역에서 표현한 것입니다.

🔄 블록선도 구성

이 회로의 블록선도는 입력 I(s)에 임피던스 블록 Z(s) = R + sL이 적용되어 출력 E(s)를 생성하는 형태입니다:

I(s) ──▶ [R + sL] ──▶ E(s)

이런 형태의 블록선도가 보기 1번에 해당되어, 정답은 1번이 되는 것이며, 해설의 정답률이 66%인 것으로 보아 다수 응시자가 이 표현을 잘 이해한 것으로 풀이됩니다.

🎯 감쇠진동과 제동비 ζ 설명

s2+2ζωns+ωn2=0s^2 + 2ζω_n s + ω_n^2 = 0

여기서:

  • s: 라플라스 변수

  • ζ (제동비): 감쇠의 정도를 나타내는 비율

  • ωₙ: 고유 진동수

🔍 감쇠 유형별 제동비 ζ의 범위

제동 유형ζ의 범위설명
과제동 (과감쇠)$$ζ > 1$$실근 2개 → 과도하게 빠른 감쇠, 진동 없음
임계제동 (임계감쇠)$$ζ = 1$$실근 중복 → 가장 빠르게 안정, 진동 없음
부족제동 (감쇠진동)$$0 < ζ < 1$$복소근 → 진동하면서 점차 안정됨 (이 경우 정답)
무제동 (순진동)$$ζ = 0$$$$s^2 + ω_n^2 = 0$$ → 진동만 계속됨, 감쇠 없음
발산$$ζ < 0$$$$s^2 -$$ 형태 → 시스템이 불안정하게 동작

✅ 정답은 4번: $$0 < ζ < 1$$

이때 시스템은 감쇠 진동(underdamped) 상태로, 진동은 하지만 점점 진폭이 작아져서 안정화됩니다. 실생활에서는 예를 들어 자동차 서스펜션 같은 데서 적절한 감쇠진동을 유지하는 것이 중요하죠—너무 빠르거나 느리면 승차감이 나빠지니까요.

63. 다음 그림의 전달함수 Y(z)/R(z)는 다음 중 어느 것인가?(정답률:41%)      
     1. G(z)z
     2. G(z)z-1
     3. G(z)Tz-1
     4. G(z)Tz

     정답 : [2]

해설

🎯 문제 해설 요약 및 설명

이 문제는 이산 시스템에서의 전달함수 분석에 관한 것으로, Y(z)/R(z)=G(z)z−1Y(z)/R(z) = G(z) z^{-1} 형태의 전달함수가 나온다는 점에서, 지연(delay) 요소가 포함된 구조임을 나타냅니다.

🔍 핵심 개념

  • G(z)G(z): 시스템의 기본 전달함수

  • z−1z^{-1}: 단위 지연 요소로, 한 시간 간격(혹은 샘플링 주기 T)만큼 출력을 뒤로 밀리게 하는 블록

  • 즉, 전체 시스템은 입력 R(z)R(z)가 G(z)G(z)를 거친 후, 한 샘플 지연되어 Y(z)Y(z)로 출력되는 구조입니다.

🧠 전달함수 형태 설명

  • Y(z)=G(z)⋅z−1⋅R(z)Y(z) = G(z) \cdot z^{-1} \cdot R(z)

  • → 이는 다음과 같은 블록선도 구조로 표현됩니다:

R(z) ──▶ [G(z)] ──▶ [z⁻¹] ──▶ Y(z)

따라서,

전달함수: $$\frac{Y(z)}{R(z)} = G(z)z^{-1}$$

✅ 선택지 분석

선택지 번호표현해설
1G(z)z미래 지연, 비정상적 표현
2 ✅G(z)z⁻¹정상적인 1샘플 지연 표현, 정답
3G(z)Tz⁻¹T는 샘플링 주기, 표현 오류
4G(z)TzT 곱 + z: 구조적으로 부적합

🧩 이 문제는 정답률이 41%로 다소 낮았는데요, 아마도 "지연 요소"를 어떻게 해석하느냐가 주요 포인트였던 것 같아요. 이산 제어에서의 단위 지연과 연속 제어에서의 지연은 조금 다른 방식으로 다뤄지니, 혼동하지 않도록 해둘 필요가 있습니다.

64. 일정 입력에 대해 잔류편차가 있는 제어계는?(정답률:65%)

     2. 적분 제어계

     4. 비례 적분 미분 제어계

     정답 : [1]

해설

🛠️ 제어계의 잔류편차 관련 해설

이 문제는 제어 시스템에서 정상상태 오차(steady-state error), 즉 잔류편차가 존재하는지 여부를 묻는 문제입니다. 시스템에 일정한 입력(예: 계단 입력)이 주어졌을 때, 출력이 입력과 완전히 일치하지 않는 상태를 잔류편차라고 합니다.

🔍 각 제어 방식에 따른 특징

제어 방식잔류편차 존재 여부설명
비례 제어 (P)✅ 있음오차에 비례하여 제어하지만 오차를 완전히 제거하지는 못함
적분 제어 (I)❌ 없음오차가 쌓이면 작동해 잔류편차를 없앰
비례-적분 제어 (PI)❌ 없음비례 + 적분으로 오차 제거 가능
비례-적분-미분 제어 (PID)❌ 없음PI에 미분을 추가하여 과도응답까지 개선

✅ 정답은 1번: 비례 제어계

적분 제어(I)가 포함된 시스템은 정상상태 오차를 제거합니다. 따라서 "잔류편차가 있는" 제어계를 묻는 이 문제에서는 적분 요소가 없는 비례 제어가 정답입니다.

💡 참고로, 미분 제어(D)는 과도응답 개선에는 효과적이지만 정상상태 오차를 없애는 역할은 하지 않습니다. 그래서 잔류편차 제거에는 직접적인 기여가 없습니다.

📘 제어시스템의 안정 조건 해설

이 문제는 제어 시스템의 안정성을 판단하기 위한 핵심 조건을 묻고 있습니다. 그 중심 개념은 BIBO 안정성입니다.

🔍 BIBO 안정성이란?

BIBO (Bounded Input, Bounded Output) 안정성 → 시스템이 입력이 유한(한정)일 때, 출력도 반드시 유한해야 한다는 조건

✅ 선택지 분석

번호내용해설정답 여부
1입력이 있을 때 출력이 0으로 감모든 시스템이 0으로 가는 것은 아님. 특정 조건일 뿐❌
2입력이 없을 때 출력이 무한대로 감입력이 없는데 출력이 무한이라면 불안정❌
3유한한 입력 → 무한한 출력이는 불안정 시스템의 정의❌
4유한한 입력 → 유한한 출력BIBO 안정성 그대로 반영된 조건✅ 정답

📐 안정한 시스템의 예시

  • 온도 제어 시스템: 입력 온도가 변화하더라도 출력(온도 조절 결과)은 제어 범위 내에서 유지

  • 자동차 서스펜션 시스템: 노면 변화(입력)가 있어도 출력(차체의 흔들림)이 제어 범위 안에서 움직임

간단히 정리하면: “입력 신호가 제한되어 있을 때 출력도 제한되어야 한다” → 이게 바로 시스템 안정의 본질이죠.

📉 방정식의 안정성 및 실수부 근 분석

주어진 3차 방정식은 다음과 같습니다:

s3+11s2+2s+40=0s^3 + 11s^2 + 2s + 40 = 0

이 식에서 양의 실수부를 갖는 근의 개수를 찾는 것이 문제의 핵심입니다. 해설에서는 계산기 활용과 루스-후르비츠(Routh-Hurwitz) 표를 사용하는 두 가지 방법을 소개하고 있습니다. 아래에 각 접근 방식을 정리해 볼게요.

🧠 방법 ①: 루스-후르비츠(Routh-Hurwitz) 판별법

  • 이 방법은 시스템의 극점(pole)의 실수부 부호(+, −)를 판단해 시스템 안정성과 근 개수를 추론하는 데 유용합니다.

🔸 1열 구성

방정식 계수로 루스표 작성:

행항목계산
R₁s³ 행1 (앞 계수)
R₂s² 행11
R₃s¹ 행$$\frac{-18}{11}$$
R₄s⁰ 행40

  • 1열에서 부호 변화 횟수가 시스템의 불안정성 정도를 나타냄

  • 여기서는 → 1 → 11 → -18/11 → 40 → 부호 변화 2번

✅ 따라서 실수부가 양(+)인 근이 2개입니다.

📱 방법 ②: 공학용 계산기 활용

카시오 fx-570ES PLUS 계산기의 다항식 해석 기능(EQN 모드)을 활용해 직접 근을 계산한 경우:

  • 입력값: 1, 11, 2, 40

  • 결과: 실근 중 1개는 음수(-11), 나머지 2개는 양의 실수근

  • 역시 정답은 2개

68. 논리식 를 간략화한 것은?(정답률:73%)      
     1. x + y
     2.
     3.
     4.

     정답 : [2]

해설

🧠 논리식 간략화 해설

주어진 식을 부울 대수(Boolean Algebra)를 사용해 간단히 정리하는 문제입니다. 식은 다음과 같이 주어져 있습니다:

L=x‾⋅y‾+x‾⋅y+x⋅yL = \overline{x} \cdot \overline{y} + \overline{x} \cdot y + x \cdot y

📘 Step-by-step 간략화 과정

부울 대수의 기본 공식과 특성을 활용합니다:

  1. 분배법칙과 항의 결합을 이용하여 정리:

    L = (¬x ∧ ¬y) + (¬x ∧ y) + (x ∧ y)

  2. 앞 두 항을 묶기:

    L = ¬x ∧ (¬y + y) + x ∧ y
   → ¬x ∧ 1 + x ∧ y  (왜냐하면 ¬y + y = 1)

  3. 논리곱 처리:

    L = ¬x + x ∧ y

  4. 또다시 항 결합을 위해 분배:

    L = (¬x + x) ∧ (¬x + y)
   → 1 ∧ (¬x + y)  (왜냐하면 ¬x + x = 1)
   → ¬x + y

✅ 최종 간략화된 식

L=x‾+yL = \overline{x} + y

즉, 정답 2번이 옳습니다 ✅

법칙내용
항등법칙$$x + \overline{x} = 1$$
소멸법칙$$x \cdot \overline{x} = 0$$
분배법칙$$x \cdot (y + z) = x \cdot y + x \cdot z$$
멱등법칙$$x + x = x, \quad x \cdot x = x$$

전달함수 계산에는 다음과 같은 두 가지 요소가 사용됩니다:

  • 정방향 경로의 합산: 입력 R(s)에서 출력 C(s)까지 직접적으로 연결된 경로상의 블록들의 곱의 합

  • 되먹임(피드백) 경로의 보정: 전체 전달함수의 분모는 되먹임 경로로 인해 보정됨

전형적인 되먹임 구조의 전달함수는 다음과 같은 형태입니다:

C(s)R(s)=정방향 경로의 합1−되먹임 경로의 곱의 합\frac{C(s)}{R(s)} = \frac{\text{정방향 경로의 합}}{1 - \text{되먹임 경로의 곱의 합}}

🧠 문제 속 값 대입

  • 정방향 경로:

    • 첫 경로: 2 × 4 = 8

    • 두 번째 경로: 5 × 4 = 20 ⇒ 총합: 8 + 20 = 28

  • 되먹임 경로:

    • 피드백 블록이 –4임 ⇒ 분모: $$1 - (-4) = 1 + 4 = 5$$

따라서 전달함수는:

C(s)R(s)=285\frac{C(s)}{R(s)} = \frac{28}{5}

✅ 정답: 3번 (28/5)

정답률이 74%로 높은 편이며, 이는 기본적인 블록선도 전달함수 구조만 이해하면 쉽게 접근할 수 있다는 뜻입니다.

📐 진폭 A 및 위상각 θ 해설

이 문제는 시스템 전달함수의 주파수 응답을 평가하여 진폭(Amplitude)과 위상각(Phase)을 분석하는 것입니다. 문제에 명시된 전달함수는 다음과 같이 주어졌다고 가정합니다:

H(jω)=K−ω2+jωH(j\omega) = \frac{K}{- \omega^2 + j\omega}

🔍 ① 진폭 (Magnitude, A)

진폭은 전달함수의 복소수 크기에 해당하며, 주파수 ω가 무한대로 갈 때 어떻게 수렴하는지를 파악해야 합니다.

  • 분모: $$ -\omega^2 + j\omega $$

  • 절댓값 계산:

∣H(jω)∣=∣K−ω2+jω∣=∣K∣(ω2)2+(ω)2=∣K∣ωω2+1→ω→∞0|H(j\omega)| = \left| \frac{K}{-\omega^2 + j\omega} \right| = \frac{|K|}{\sqrt{(\omega^2)^2 + (\omega)^2}} = \frac{|K|}{\omega\sqrt{\omega^2 + 1}} \xrightarrow{\omega \to \infty} 0

✅ 따라서 진폭 A = 0

🔍 ② 위상각 (Phase Angle, θ)

위상각은 전달함수의 복소수 인수에 포함된 각도(Arg)이며, 분모의 복소 부분에 따라 결정됩니다.

  • 분모: $$ -\omega^2 + j\omega $$

  • 극한 계산 시:

    • 실수부: $$ -\omega^2 \to -\infty $$

    • 허수부: $$ \omega \to \infty $$

  • 따라서, 분모의 위상은:

θdenominator=tan⁡−1(ω−ω2)≈tan⁡−1(−1ω)→0 (아래쪽 방향)\theta_\text{denominator} = \tan^{-1} \left( \frac{\omega}{- \omega^2} \right) \approx \tan^{-1} \left( -\frac{1}{\omega} \right) \to 0 \text{ (아래쪽 방향)}

정확하게는 분모의 각도가 -180°, 즉 위상이 완전히 반전된 상태가 됩니다.

✅ 전달함수 전체의 위상각은 θ = -180°

✅ 최종 정답: 2번 (A = 0, θ = −180°)

📌 기억해둘 점

  • 진폭은 0: 분모가 무한대가 되면서 전체 전달함수가 0에 가까워짐

  • 위상은 −180°: 분모가 실수 축 음수 방향으로 극단적으로 치우치므로

이러한 주파수 응답 해석은 보드선도(Bode plot)나 위상 마진 같은 고급 제어 분석에도 연결될 수 있어요. 여기에 이어서 보드선도나 고주파 응답에 관심 있으세요? 🎛️

I=VZ=100133.47≈0.749 AI = \frac{V}{Z} = \frac{100}{133.47} \approx 0.749\,\text{A}

📡 무손실 전송선로의 정상상태 해설

이 문제는 고주파 및 통신공학에서의 전송선로(Transmission Line) 이론 중에서, 무손실 조건에서의 물리적 특성을 묻는 것입니다. 해설에 따라 정답은 2번(특성 임피던스에 대한 설명이 틀림)이고, 그 이유를 구체적으로 풀어볼게요:

🧪 ① 무손실 전송선로란?

무손실 선로란 다음의 조건을 만족하는 전송선로입니다:

  • 저항 R = 0 (선로의 저항 없음)

  • 컨덕턴스 G = 0 (절연 손실 없음)

이 조건 하에서는 에너지 손실 없이 신호가 전달되며 전송선로의 핵심 파라미터들이 간단하게 계산됩니다.

⚙️ 무손실 선로의 주요 파라미터 정리

파라미터정의 및 식설명
전파정수 γ$$ γ = α + jβ = \sqrt{(R + jωL)(G + jωC)} $$→ R=G=0이면 $$ γ = jω\sqrt{LC} $$
특성 임피던스 Z₀$$ Z_0 = \sqrt{\frac{R + jωL}{G + jωC}} $$→ R=G=0이면 $$ Z_0 = \sqrt{\frac{L}{C}} $$
진행파 속도 v$$ v = \frac{1}{\sqrt{LC}} $$→ 유한한 속도로 진행
감쇠정수 α$$ α = 0 $$→ 무손실이므로 감쇠 없음
위상정수 β$$ β = ω\sqrt{LC} $$→ 위상 전달 속도에 영향

❌ 2번이 틀린 이유

정답 선택지 2번은 특성 임피던스에 대해 틀린 설명을 했기 때문입니다.

  • 일반적인 특성 임피던스 공식: $$ Z_0 = \sqrt{\frac{R + jωL}{G + jωC}} $$

  • 무손실 조건에서는 $$ R = 0 $$, $$ G = 0 $$ 이므로:

그러므로, 특성 임피던스를 잘못 계산하거나 잘못 표시한 것이 문제에서 틀린 설명이라는 의미입니다.

✅ 나머지 선택지는 옳은 설명

  • 1번: $$ γ = jω\sqrt{LC} $$ → 옳음

  • 3번: $$ v = \frac{1}{\sqrt{LC}} $$ → 옳음

  • 4번: $$ α = 0, β = ω\sqrt{LC} $$ → 옳음

📐 파형의 라플라스 변환 해설

이 문제는 시간 영역에서 정의된 f(t)를 라플라스 변환하여 주파수 영역 함수 F(s)로 바꾸는 문제입니다. 주어진 파형의 수식은 다음과 같습니다:

f(t)=12t⋅u(t)−12(t−4)⋅u(t−4)−2⋅u(t−4)f(t) = \frac{1}{2} t \cdot u(t) - \frac{1}{2}(t-4) \cdot u(t-4) - 2 \cdot u(t-4)

여기서:

  • u(t)u(t): 단위 계단 함수(Unit Step Function) — 해당 시점부터 함수가 활성화됨

  • 표현은 부분적으로 정의된 함수이며, 구간별로 다르게 작동하는 파형임

🔄 Step-by-Step: 라플라스 변환

각 항목을 따로 라플라스 변환한 후 합산합니다.

✏️ 첫 항: 12t⋅u(t)\frac{1}{2} t \cdot u(t)

  • 라플라스 변환: $$ \mathcal{L}\left\{ \frac{1}{2}t \right\} = \frac{1}{2} \cdot \frac{1}{s^2} $$

✏️ 두 번째 항: 12(t−4)⋅u(t−4)\frac{1}{2}(t-4) \cdot u(t-4)

  • 시간 지연 항: $$ \mathcal{L}\left\{ (t-4) \cdot u(t-4) \right\} = \frac{e{-4s}}{s2} $$

  • → 전체 항은: $$ \frac{1}{2} \cdot \frac{e{-4s}}{s2} $$

✏️ 세 번째 항: 2⋅u(t−4)2 \cdot u(t-4)

  • 라플라스 변환: $$ 2 \cdot \frac{e^{-4s}}{s} $$

✅ 최종 결과

이 세 항을 조합하면 다음과 같습니다:

F(s)=12s2−12s2e−4s−2se−4sF(s) = \frac{1}{2s^2} - \frac{1}{2s^2} e^{-4s} - \frac{2}{s} e^{-4s}

F(s)=12s2(1−e−4s)−2se−4sF(s) = \frac{1}{2s^2}(1 - e^{-4s}) - \frac{2}{s} e^{-4s}

  • W₂ = 1400W

👉 이 값은 두 개의 전력계(Wattmeter)가 각각 측정한 유효전력(P)의 일부이며, 전체 유효전력 P = W₁ + W₂ = 2100W입니다.

🧮 피상전력 S 계산 공식 (2전력계법)

피상전력(S)을 구하는 공식은:

S=2W12+W22−W1⋅W2S = 2 \sqrt{W_1^2 + W_2^2 - W_1 \cdot W_2}

이 식은 평형 3상 시스템에서 유효전력과 무효전력의 관계를 간접적으로 반영해줍니다.

📐 대입 및 계산:

  • $W_1 = 700$

  • $W_2 = 1400$

계산:

S=27002+14002−700⋅1400=2490000+1960000−980000=21470000=2⋅1212.4≈2424.8 VAS = 2 \sqrt{700^2 + 1400^2 - 700 \cdot 1400} \\ = 2 \sqrt{490000 + 1960000 - 980000} \\ = 2 \sqrt{1470000} \\ = 2 \cdot 1212.4 \approx 2424.8\, \text{VA}

✅ 따라서 최종 피상전력은 약 2425 VA이고, 정답은 1번입니다.

📌 추가 개념: 역률 계산도 가능

  • 유효전력 P=W1+W2=2100 WP = W_1 + W_2 = 2100\,W

cos⁡θ=PS=21002425≈0.866\cos\theta = \frac{P}{S} = \frac{2100}{2425} \approx 0.866

이 값은 이상적인 3상 역률(약 cos⁡30∘=0.866\cos 30^\circ = 0.866)와 동일하므로, 시스템이 평형이고 위상차가 30도임을 암시합니다.

⚡ 기본 개념

  • 일반적인 정현파의 실효값: $$ \frac{I_m}{\sqrt{2}} $$

  • 반파 정류된 정현파의 실효값: $$ I_{\text{rms}} = \sqrt{ \frac{1}{2\pi} \int_{0}^{\pi} I_m^2 \sin^2(\theta) \, d\theta } $$

🔸 계산

Irms=Im22π∫0πsin⁡2(θ) dθ=Im22π⋅π2=Im24=Im2\begin{align*} I_{\text{rms}} &= \sqrt{ \frac{I_m^2}{2\pi} \int_{0}^{\pi} \sin^2(\theta) \, d\theta } \\ &= \sqrt{ \frac{I_m^2}{2\pi} \cdot \frac{\pi}{2} } = \sqrt{ \frac{I_m^2}{4} } = \frac{I_m}{2} \end{align*}

✅ 따라서 실효값은 Im2\frac{I_m}{2}

📊 선택지 분석

번호표현해설

📐 파고율(Peak Factor) 해설

이 문제는 주어진 파형의 파고율(Peak Factor)을 묻는 것으로, 파고율은 다음과 같이 정의됩니다:

파고율=최대값실효값\text{파고율} = \frac{\text{최대값}}{\text{실효값}}

🔍 주어진 조건: 구형 반파 정류파

문제에서 언급된 파형은 구형 반파 정류파(half-wave sinusoidal signal)입니다. 이 파형의 최대값은 ImI_m이고, 실효값은 아래와 같이 계산됩니다:

  • 최대값: ImI_m

  • 실효값: $$ I_{\text{rms}} = \frac{I_m}{\sqrt{2}} $$

✅ 계산

파고율 계산:

ImIm/2=2\frac{I_m}{I_m / \sqrt{2}} = \sqrt{2}

즉, 파고율 = 2\sqrt{2}

📌 선택지 분석

번호표현해석
11파고율이 1인 경우는 정현파가 아닌 평탄파형
212\frac{1}{\sqrt{2}}실효값이 최대값보다 더 큰 비현실적 경우
3 ✅2\sqrt{2}정답! 구형 정현파의 전형적인 파고율
43\sqrt{3}일부 임의파형에서 나올 수 있음

🎯 결론

  • 정답은 3번 2\sqrt{2}

  • 구형파, 정현파 모두에서 자주 등장하는 기본 파고율입니다.

📘 4단자 정수(ABCD Parameter) 해설

이 문제는 2포트 네트워크(4단자 회로)의 행렬 분석을 다루고 있습니다. 주어진 상황은:

  • 10Ω 저항

  • 권수비가 10:1인 결합회로(변압기)

이를 통해 2포트 시스템의 ABCD 정수를 구하는 문제입니다.

🔍 기본 개념: 4단자 정수란?

4단자 정수(ABCD 파라미터)는 다음의 선형 방정식을 기반으로 합니다:

[V1I1]=[ABCD]⋅[V2−I2]\begin{bmatrix} V_1 \\ I_1 \end{bmatrix} = \begin{bmatrix} A & B \\ C & D \end{bmatrix} \cdot \begin{bmatrix} V_2 \\ - I_2 \end{bmatrix}

이때 A, B, C, D는 각각 입력전압·전류와 출력전압·전류 사이의 관계를 설명하는 상수입니다.

🧠 회로 구성 분석

🔸 1단계: 저항 10Ω의 ABCD 정수

저항 R=10 ΩR = 10\,\Omega의 ABCD 매트릭스:

[1R01]=[11001]\begin{bmatrix} 1 & R \\ 0 & 1 \end{bmatrix} = \begin{bmatrix} 1 & 10 \\ 0 & 1 \end{bmatrix}

🔸 2단계: 이상적인 변압기 (권수비 10:1)

권수비 n=10n = 10, 입력 측이 10권, 출력 측이 1권 → ABCD 매트릭스:

[n001n]=[10000.1]\begin{bmatrix} n & 0 \\ 0 & \frac{1}{n} \end{bmatrix} = \begin{bmatrix} 10 & 0 \\ 0 & 0.1 \end{bmatrix}

🔧 전체 회로의 ABCD 행렬 계산

회로는 저항 → 변압기 순서이므로, 행렬의 곱을 순서대로 적용:

[A,B;C,D]=[11001]⋅[10000.1]=[10100.1][A,B;C,D] = \begin{bmatrix} 1 & 10 \\ 0 & 1 \end{bmatrix} \cdot \begin{bmatrix} 10 & 0 \\ 0 & 0.1 \end{bmatrix} = \begin{bmatrix} 10 & 1 \\ 0 & 0.1 \end{bmatrix}

✅ 최종 정답: 4번

  • A = 10

  • B = 1

  • C = 0

  • D = 1/10

💡 이런 행렬은 신호 전송 시스템, 고주파 회로 설계, 제어 시스템의 모듈화 등에서 널리 활용됩니다.

🔌 RC 회로에서 스위치 작동 시 전류의 형태

이 문제는 RC 회로(저항 R과 커패시터 C의 직렬 회로)에서 스위치를 닫는 순간의 전류 변화 양상을 묻는 것입니다. 주어진 조건은 초기조건이 0이라는 점이며, 이때 전류는 어떤 형태로 시간에 따라 변화하는지를 선택지 중에서 고르는 문제입니다.

⚙️ RC 회로의 기본 동작 원리

  • 커패시터는 처음에는 도선처럼 행동하고, 시간이 지날수록 전압이 쌓이며 전류가 감소함.

  • 저항(R)과 커패시터(C)로 구성된 회로에서 전류는 지수 함수적으로 감소하는 형태.

▶ 시간 t=0에서 스위치 ON일 때:

  • 커패시터에 축적된 전하가 없으므로 전류는 최대로 흐름

  • 이후 커패시터에 전압이 점점 쌓여 전류는 감소

  • 결과적으로 전류 i(t)i(t)는 아래와 같은 감쇠 함수 형태:

i(t)=VR⋅e−t/RCi(t) = \frac{V}{R} \cdot e^{-t/RC}

  • 여기서 VV: 인가 전압, RR: 저항, CC: 커패시턴스

✅ 선택지 분석

번호설명해설
1불변전류RC 회로의 전류는 시간에 따라 변함 → ❌
2진동전류진동은 RLC 회로에서 가능함 → ❌
3증가함수전류는 감소하지 증가하지 않음 → ❌
4 ✅감쇠함수RC 회로에서 전형적인 전류 응답 형식 → ✅ 정답

🧠 보충 정리

  • RL 회로: 전류가 점차 증가 → i(t)=VR(1−e−tL/R)i(t) = \frac{V}{R}(1 - e^{-tL/R})

  • RC 회로: 전류가 점차 감쇠 → i(t)=VRe−t/RCi(t) = \frac{V}{R}e^{-t/RC}

  • RLC 회로: 조건에 따라 진동하거나 불변일 수 있음 (저항, 인덕턴스, 커패시턴스 조합에 따라 다름)

결국 이 문제에서 RC 회로의 핵심은 커패시터에 전하가 축적됨으로써 전류가 점차 줄어드는 감쇠 함수적인 특성을 이해하는 것입니다.

🔎 중첩의 정리로 전류 계산 해설

이 문제는 회로에서 특정 저항 R에 흐르는 전류 I를 구하는 것이며, 중첩의 정리(Superposition theorem)를 사용하여 해석합니다.

🧠 중첩의 정리 개요

선형 회로에서 여러 개의 전원이 있을 경우, 각 전원에 의한 개별 효과(전류 또는 전압)를 계산한 후 전체 응답은 그 합으로 구합니다.

⚙️ 문제 접근 방식

  1. 전압원을 단락(전압원 OFF 상태)

  2. 전류원을 개방(전류원 OFF 상태)

  3. 각 경우에서의 전류를 I′, I″로 계산

  4. 방향에 유의하여 합산: $$ I = I' - I'' $$

🔢 해설 요약

✔️ 전류원 개방 → 전압원만 작동

  • 등가 저항을 계산 → 총 저항: 3Ω

  • 전체 전류: I전체=VR=63=2AI_\text{전체} = \frac{V}{R} = \frac{6}{3} = 2A

  • 해당 저항 R에 흐르는 전류는 1A

✔️ 전압원 단락 → 전류원만 작동

  • 병렬 회로에서 분류된 전류 값 → 3A

  • 방향이 I′와 반대이므로 −3A

✅ 최종 계산

I=I′+(−I′′)=1−3=−2 AI = I' + (−I'') = 1 − 3 = −2\,\text{A}

✅ 정답: 2번 −2A

이 문제는 중첩의 정리와 방향성 이해가 핵심입니다. 정답률이 51%라는 건 절반 이상이 개념을 잘 따라갔지만, 회로에서 전원 OFF 상태 처리 방식에서 실수가 많았다는 뜻이에요.

🔧 3상 교류발전기의 대칭분과 정상분 전압식 해설

이 문제는 대칭분 해석(Symmetrical Component Analysis)을 기반으로 3상 교류발전기에서 정상분 전압 V1V_1을 구하는 기본식을 묻는 내용입니다.

⚙️ 대칭분 해석의 기초

3상 교류 시스템에서 발생하는 불평형 전류와 전압을 해결하기 위해, 이를 정상분(I₁, V₁), 역상분(I₂, V₂), 영상분(I₀, V₀)로 나누어 분석합니다.

각 대칭분은 발전기 내부에서 다음과 같은 관계식을 갖습니다:

대칭분 종류기본식설명
영상분V0=−Z0I0V_0 = -Z_0 I_0전원이 없고 임피던스만 존재
정상분V1=Ea−Z1I1V_1 = E_a - Z_1 I_1정상 기전력과 정상 임피던스 포함
역상분V2=−Z2I2V_2 = -Z_2 I_2전원이 없고 임피던스만 존재

📌 추가 개념: 왜 Ea만 포함되나요?

  • 정상분 회로에서는 균형 잡힌 기전력 Ea, Eb, Ec가 동일하게 영향을 미침

  • 따라서 단상처럼 EaE_a 한 개만 대표로 사용해 표현 가능

  • 정상분은 3상 전력의 균형된 구성 요소라서 기전력이 반영되지만, 영상/역상에서는 기전력이 포함되지 않습니다

대칭분 해석은 고장 해석, 보호 계전기 설계, 전력계통 안정도 분석 등에 널리 쓰이며, 특히 지락 및 단락 고장 시 각 분의 흐름을 따져보는 데 유용합니다.
