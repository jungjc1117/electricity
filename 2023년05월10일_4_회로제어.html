⚡ 설명: 영점과 극점 분석

  • 영점(zero): 분모가 아닌 분자를 0으로 만드는 ss의 값 즉, s+7=0⇒s=−7s + 7 = 0 \Rightarrow s = -7 그리고 s+8=0⇒s=−8s + 8 = 0 \Rightarrow s = -8

따라서 이 함수의 영점은 -7과 -8, 극점은 -6입니다.

📘 문제 개요 대칭좌표법(Symmetrical Component Method)은 전력시스템에서 3상 불평형을 분석하는 데 사용되는 방법입니다. 불평형률은 시스템이 얼마나 정상적인 3상 상태에서 벗어났는지를 나타내는 지표로, 정상분에 대한 역상분이나 영상분의 비율로 계산됩니다.

⚙️ 용어 정리

  • 정상분(Positive-sequence): 정방향 회전의 정상적인 전력 구성 요소

  • 역상분(Negative-sequence): 반대방향 회전의 불평형 구성 요소

  • 영상분(Zero-sequence): 세 상이 동일한 위상과 크기를 갖는 구성 요소

🧮 불평형률 공식

불평형률은 일반적으로 다음과 같이 표현됩니다:

불평형률=역상분정상분×100[%]\text{불평형률} = \frac{\text{역상분}}{\text{정상분}} \times 100 \quad [\%]

※ 영상분을 사용하는 경우도 있지만, 가장 일반적인 불평형률 지표는 역상분 기준입니다.

✅ 보기 비교

  • ①: 영상분정상분×100\frac{\text{영상분}}{\text{정상분}} \times 100 → 영상 기반, 비일반적

  • ②: 정상분역상분×100\frac{\text{정상분}}{\text{역상분}} \times 100 → 역상분이 분모에 있어야 함, 역전됨

  • ③: 영상분정상분×100\frac{\text{영상분}}{\text{정상분}} \times 100 → ①과 동일

  • ④: 역상분정상분×100\frac{\text{역상분}}{\text{정상분}} \times 100 → ✅ 가장 정석적인 표현

🎯 최종 정답: ④번

정상분과 역상분의 비율을 기준으로 불평형을 측정하는 것이 대칭좌표법에서 가장 일반적으로 사용되는 방식입니다.

이 문제는 콘덴서가 포함된 저항-콘덴서(RC) 직렬회로 또는 저항-유도(RL) 회로에서의 역률(Power Factor)을 묻는 것으로 보입니다. 제시된 보기의 구조를 통해 역률 계산 공식이 포함된 것으로 판단됩니다.

⚙️ 역률 공식 정리

역률(PF)은 일반적으로 다음 공식으로 계산됩니다:

PF=cos⁡θ=RZ\text{PF} = \cos\theta = \frac{R}{Z}

여기서:

  • RR: 저항

  • ZZ: 합성 임피던스의 크기

  • θ\theta: 전압과 전류의 위상차

🔍 역률의 수식 표현

주어진 보기들은 임피던스의 성분을 활용한 표현으로, 다음과 같은 구조일 것으로 예상됩니다:

Z=R2+(XC)2=1+(ωRC)2⇒PF=11+(ωRC)2Z = \sqrt{R^2 + (X_C)^2} = \sqrt{1 + (\omega RC)^2} \quad \Rightarrow \quad \text{PF} = \frac{1}{\sqrt{1 + (\omega RC)^2}}

따라서:

  • 역률(PF) = 11+(ωRC)2\displaystyle \frac{1}{\sqrt{1 + (\omega RC)^2}}

✅ 보기와 비교

  • ①: 1+(ωRC)21 + (\omega RC)^2 → 전체적인 임피던스 제곱

  • ②: 1+(ωRC)2\sqrt{1 + (\omega RC)^2} → 임피던스 크기 표현

  • ③: 11+(ωRC)2\frac{1}{\sqrt{1 + (\omega RC)^2}} → ✅ 정확히 역률 표현!

  • ④: 보기가 완성되지 않음 (불완전)

🎯 최종 정답: ③번

해당 회로는 R과 C가 직렬로 구성된 RC회로로 추정되며, 위상지연이 발생하고 역률은 감소하게 됩니다. 보기 ③의 식은 위 공식과 완전히 일치하므로 정답입니다.

⚙️ 실효값 계산 공식

비정현파일 때, 개별 성분의 RMS 값을 다음 방식으로 처리합니다:

Vrms=V12+V22V_{\text{rms}} = \sqrt{V_1^2 + V_2^2}

여기서:

  • V1=A12=17.172≈12.14V_1 = \frac{A_1}{\sqrt{2}} = \frac{17.17}{\sqrt{2}} \approx 12.14

  • V2=A22=7.072≈5.00V_2 = \frac{A_2}{\sqrt{2}} = \frac{7.07}{\sqrt{2}} \approx 5.00

그럼 전체 실효값:

Vrms=12.142+5.002=147.37+25.00=172.37≈13.13 VV_{\text{rms}} = \sqrt{12.14^2 + 5.00^2} = \sqrt{147.37 + 25.00} = \sqrt{172.37} \approx 13.13\ \text{V}

🔍 복원된 시험 자료 기준

복원된 CBT 시험 자료에 따르면, 이 문제의 정답은 ②번, 11.2V로 제시되어 있습니다.

Vrms=(17.1722)+(7.0722)≈147.37+25.00=172.37≈13.13 VV_{\text{rms}} = \sqrt{ \left( \frac{17.17^2}{2} \right) + \left( \frac{7.07^2}{2} \right) } \approx \sqrt{147.37 + 25.00} = \sqrt{172.37} \approx 13.13\ \text{V}

① 프로세스 제어 (Process Control) 연속적으로 기준값을 유지하며 제어—✔ 추치제어에 해당됨

제어 방식 설명
④ 프로그램제어 (Programmed Control) 기준값이 시간에 따라 고정된 스케줄에 따라 미리 설정됨 → 🔻 단순한 시간 순차 제어로 추종 제어와 다름

따라서 프로그램제어는 추치제어가 아닙니다.

이 문제는 자동제어에서 단위 피드백 구조를 갖는 제어계의 특성방정식(Characteristic Equation)을 찾는 문제입니다.

  • 폐루프 전달함수:

T(s)=G(s)H(s)1+G(s)H(s)T(s) = \frac{G(s)H(s)}{1 + G(s)H(s)}

여기서, 특성방정식은 폐루프 시스템의 분모에 해당합니다:

1+G(s)H(s)=01 + G(s)H(s) = 0

📌 보기 분석

보기 번호 표현 의미 정답 여부
① G(s)H(s)=1G(s)H(s) = 1 ❌ 역이득 조건, 특성방정식 아님
② G(s)H(s)=−1G(s)H(s) = -1 ✅ 특성방정식 1+G(s)H(s)=01 + G(s)H(s) = 0에서 도출
