📊 문제 설명 및 해설 요약

이 문제는 전달함수로 표현된 제어 시스템에서 직류 이득 (DC gain)을 구하는 것을 묻고 있습니다. 직류 이득은 입력이 정상 상태(steady state)에서 일정하게 유지될 때 시스템이 출력으로 얼마나 반응하는지를 나타내는 값입니다.

📌 핵심 개념: 직류 이득 계산 방법

  • 따라서 직류 이득 = G(0)G(0) 로 구하면 됩니다.

📐 예시 및 풀이 흐름

  • 문제에서는 전달함수의 상세 식이 생략되어 있지만, 해설에 따르면 G(0)=5G(0) = 5가 나오며, 이는 정답 선택지 4번입니다.

  • 따라서 문제의 조건상 정답은 [4] 5가 됩니다 ✅

Y = A'BC(1) + A'B'C(1)
  = A'BC + A'B'C

🔍 Step 2: 공통 인수 묶기

  • 두 항의 공통 인수는 A′CA'C

  • 식을 묶어주면: 

Y = A'C(B + B')

  • B+B′B + B' 역시 항상 참입니다. 즉, B+B′=1B + B' = 1

  • 그래서 식은: 

Y = A'C * 1 = A'C

Y = A'C

📘 문제 설명: 감쇠 진동 주파수 ω_d 계산

이 문제는 2차 제어시스템에서의 감쇠 진동 주파수 ωd\omega_d를 구하는 계산 문제입니다. 전달함수의 형태와 특성방정식의 계수를 통해 시스템의 고유진동수 ωn\omega_n와 감쇠비 ξ\xi를 추정한 뒤, 이를 이용해 ωd\omega_d를 산출합니다.

📐 기본 공식과 풀이 흐름

  • 표준 2차 전달함수 형태

G(s)=ωn2s2+2ξωns+ωn2G(s) = \frac{\omega_n^2}{s^2 + 2\xi\omega_n s + \omega_n^2}

  • 감쇠 진동 주파수 공식

ωd=ωn1−ξ2\omega_d = \omega_n \sqrt{1 - \xi^2}

🧮 문제에서 주어진 정보

  • 특성방정식:

s2+6s+25=0s^2 + 6s + 25 = 0

  • 비교해 보면:

    • 2ξωn=62\xi\omega_n = 6

    • ωn2=25⇒ωn=5\omega_n^2 = 25 \Rightarrow \omega_n = 5

    • ξ=62×5=0.6\xi = \frac{6}{2 \times 5} = 0.6

  • 따라서 감쇠 진동 주파수는:

ωd=5⋅1−0.62=5⋅0.64=5⋅0.8=4 rad/sec\omega_d = 5 \cdot \sqrt{1 - 0.6^2} = 5 \cdot \sqrt{0.64} = 5 \cdot 0.8 = 4 \,\text{rad/sec}

✅ 정답: [2] 4 rad/sec

이 값은 시스템이 과도한 진동 없이 안정적으로 응답하는데 필요한 주파수를 나타내며, 제어 시스템의 설계 품질과도 직접 연결되는 핵심 지표입니다.

필요하시면 감쇠비 변화에 따른 ωd\omega_d 값의 그래프나 해석적인 비교도 해드릴 수 있어요. 이 주제, 은근히 깊고 재미있답니다. 😄

🧠 기본 개념

  • 정상상태오차란 시스템 입력과 출력의 차이가 시간이 무한대로 갈 때 남는 오차를 의미합니다.

  • 폐루프 제어에서 이를 제거하려면 시스템이 오차를 누적해서 보상해주는 방식이 필요합니다.

📐 제어 기법 비교

제어 방식 오차 제거 응답 속도 특징
비례(P) 제어 ❌ 오차 존재 (offset) 보통 출력과 오차 비례
적분(I) 제어 ✅ 오차 제거 느림 (시간 지연 발생) 누적된 오차를 보상
미분(D) 제어 ❌ 오차 존재 빠름 (속도 향상) 오차 변화율에 반응
ON-OFF 제어 ❌ 단순 but 오차 발생 빠름 디지털적 전환 제어, 오차 미세조정 불가

✅ 정답: [2] 적분 제어

적분 제어는 출력이 입력에 정확히 도달하도록 오차를 시간에 따라 누적하여 보정합니다. 단점으로는 지연(time delay)이 발생할 수 있지만, 정상상태오차 제거에는 가장 효과적인 방식입니다.

⚙️ 기본 정의

  • 직렬 임피던스 ZZ: 단위 길이당 유도성 및 저항 성분 (보통 R+jωLR + j\omega L)

  • 병렬 어드미턴스 YY: 단위 길이당 정전용량 및 누설 성분 (보통 G+jωCG + j\omega C)

Z0=ZYZ_0 = \sqrt{\frac{Z}{Y}}

🧠 직관적 이해

  • 어드미턴스 YY는 임피던스 ZZ의 역수이므로,

Y=1Z⇒ZY=Z2⇒Z0=Z2=ZY = \frac{1}{Z} \Rightarrow \frac{Z}{Y} = Z^2 \Rightarrow Z_0 = \sqrt{Z^2} = Z

※ 단, 이건 Y = 1/Z인 경우에만 해당되는 특수 상황이고 일반적인 공식은 Z0=Z/YZ_0 = \sqrt{Z/Y}입니다.

🧮 구동점 임피던스 계산 정리

이 문제는 주어진 RLC 회로에서 입력 단자(a–b) 기준으로 구동점 임피던스를 구하는 계산 문제입니다. s-domain(라플라스 영역)으로 변환하여 분석합니다.

⚙️ 주어진 회로 정리

  • 저항(R): 1Ω → 그대로 유지

  • 인덕터(L = 2H):

    • 라플라스 변환 후: ZL=sL=2sZ_L = sL = 2s

  • 커패시터(C = \frac{1}{2}F):

    • 라플라스 변환 후: ZC=1sC=112s=2sZ_C = \frac{1}{sC} = \frac{1}{\frac{1}{2}s} = \frac{2}{s}

🔗 회로 구조 이해

  • 인덕터(2H)와 저항(1Ω)은 직렬 연결 → Z1=2s+1Z_1 = 2s + 1

  • 이 조합은 커패시터(2/s)와 병렬 연결

Ztotal=ZC⋅Z1ZC+Z1=2s⋅(2s+1)2s+(2s+1)Z_{\text{total}} = \frac{Z_C \cdot Z_1}{Z_C + Z_1} = \frac{\frac{2}{s} \cdot (2s + 1)}{\frac{2}{s} + (2s + 1)}

✏️ 최종 임피던스 계산

  1. 분자:

2s⋅(2s+1)=4s+2s\frac{2}{s} \cdot (2s + 1) = \frac{4s + 2}{s}

  1. 분모:

2s+2s+1=2+s(2s+1)s=2+2s2+ss\frac{2}{s} + 2s + 1 = \frac{2 + s(2s + 1)}{s} = \frac{2 + 2s^2 + s}{s}

  1. 전체:

Zeq=4s+22+2s2+sZ_{\text{eq}} = \frac{4s + 2}{2 + 2s^2 + s}

🔌 문제 해설: Δ결선 → V결선 전환 시 공급 전력 비교

이 문제는 3상 변압기의 Δ(델타) 결선 운전 중 한 대가 고장났을 때 V결선(오픈 델타)으로 운전할 경우의 전력 공급량 차이를 묻습니다.

📐 기본 개념 정리

항목 설명
Δ결선 운전 3대의 변압기 모두 사용 (정상 조건)
V결선 운전 변압기 2대만 사용 (한 대 고장 시 대체 운전)

🧠 수식 기반 계산

  • Δ결선 공급 전력:

PΔ=3VIP_{\Delta} = 3VI

  • V결선 공급 전력:

PV=3VIP_V = \sqrt{3}VI

  • 비율 계산:

PVPΔ×100=3VI3VI×100=33×100≈57.7%\frac{P_V}{P_{\Delta}} \times 100 = \frac{\sqrt{3}VI}{3VI} \times 100 = \frac{\sqrt{3}}{3} \times 100 \approx 57.7\%

해설

🔍 교류 브리지 회로의 평형 조건 요약 설명

이 문제는 교류 브리지 회로가 평형 상태에 도달하기 위한 조건을 묻는 전형적인 회로 해석 문제입니다. 교류 성분(인덕터와 커패시터)이 포함된 브리지 회로에서도 직류 브리지 회로의 분석 원리를 동일하게 적용할 수 있습니다 — 단, 임피던스를 복소수로 표현해야 한다는 차이가 있습니다.

⚙️ 기본 원리: 브리지 회로 평형 조건

브리지 회로가 평형이 되기 위해서는 양 대각선 상에 있는 임피던스들의 곱이 같아야 합니다:

Z1⋅Z4=Z2⋅Z3Z_1 \cdot Z_4 = Z_2 \cdot Z_3

직류의 경우 각 저항값을 곱해 비교하고, 교류에서는 각 요소의 임피던스 Z를 곱해서 비교합니다.

📐 문제 적용: 회로 요소의 임피던스 표현

  • 인덕터 L의 임피던스:

ZL=jωLZ_L = j\omega L

  • 커패시터 C의 임피던스:

ZC=1jωCZ_C = \frac{1}{j\omega C}

  • 저항 R₁, R₂: 그대로 R1,R2R_1, R_2

🔍 평형 조건 계산

브리지 회로의 구성상, 아래와 같은 평형 조건이 유도됩니다:

jωL=R1R2⋅jωCj\omega L = R_1 R_2 \cdot j\omega C

→ 양변에서 jωj\omega 약분하면:

L=R1R2CL = R_1 R_2 C

✅ 정답: [3] L = R₁R₂C

이는 교류 브리지 회로의 평형을 위한 전형적인 조건식으로, 인덕턴스 LL과 커패시턴스 CC, 그리고 저항값 R1,R2R_1, R_2 간의 관계가 명확히 나타납니다.

해설

📘 라플라스 변환의 기본 공식: f(t)=tnf(t) = t^n

이 문제는 함수 f(t)=tnf(t) = t^n의 라플라스 변환을 구하는 것을 묻고 있습니다.

🧠 핵심 공식

함수 tnt^n의 라플라스 변환은 다음과 같습니다:

L{tn}=n!sn+1(단, n은 정수)\mathcal{L}\{t^n\} = \frac{n!}{s^{n+1}} \quad\text{(단, \( n \)은 정수)}

  • L{⋅}\mathcal{L} \{ \cdot \}: 라플라스 변환

  • n!n!: n의 팩토리얼

  • ss: 복소 주파수 변수

🧮 예시 적용

예: f(t)=t3f(t) = t^3

L{t3}=3!s3+1=6s4\mathcal{L}\{t^3\} = \frac{3!}{s^{3+1}} = \frac{6}{s^4}

→ 보기 3번: n!sn+1\frac{n!}{s^{n+1}}이 정답 🎯

✅ 정답: [3] n!sn+1\frac{n!}{s^{n+1}}

이 공식은 시간 도함수와 적분 연산이 포함된 회로해석, 제어공학 등에서 매우 자주 등장하는 기본 라플라스 변환 중 하나입니다.
