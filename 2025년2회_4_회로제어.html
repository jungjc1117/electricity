📊 문제 설명 및 정답 해설

문항 61번은 제어공학에서의 "근궤적(root locus)"과 시스템 안정성에 대한 이해를 평가하는 문제입니다.

📌 문제 요약

  • 주어진 조건: 근궤적이 s-jω축과 교차한다

  • 질문: 이때 폐루프 제어계는 어떤 상태인가?

📘 개념 정리

근궤적(root locus)은 폐루프 시스템의 극점이 개루프 이득 KK의 변화에 따라 복소평면 상에서 어떻게 움직이는지를 나타냅니다.

복소평면의 s-jω축(허수축)과 근궤적의 교차 여부는 시스템 안정성과 밀접한 관련이 있습니다.

  • s평면에서 우측 (Re(s) > 0): 불안정한 영역

  • s평면에서 좌측 (Re(s) < 0): 안정한 영역

  • 허수축(s = jω)에서의 교차: 임계상태(Critical or Marginally Stable)

즉, 근궤적이 허수축과 교차한다는 것은 극점이 실수가 아니라 허수부를 가지는 위치에 존재하며, 시스템이 임계상태임을 의미합니다.

✅ 정답

③ 임계상태이다.

원한다면 다른 근궤적 사례나 안정성과 관련된 제어계 모델도 함께 살펴볼 수 있어요. 더 이어가 볼까요? 🚀

📈 문제 해설: 분지점(breakaway point) 찾기

이 문제는 근궤적(root locus) 상에서의 분지점(breakaway point)을 찾는 전형적인 제어공학 문제입니다. 근궤적은 폐루프 극점의 위치 변화를 시각적으로 나타내며, 분지점은 두 극점이 서로 가까워졌다가 실수축을 따라 반대 방향으로 갈라지는 지점을 말합니다.

✅ 주어진 전달함수

G(s)H(s)=Ks(s+1)(s+7)G(s)H(s) = \frac{K}{s(s+1)(s+7)}

이 시스템의 개루프 극점은:

  • s=0s = 0

  • s=−1s = -1

  • s=−7s = -7

📌 분지점 계산 절차

분지점은 실수축 상에서 극점 간의 근궤적이 갈라지는 지점으로, 일반적으로 극점들 사이에서 발생합니다. 이 지점은 dKds=0\frac{dK}{ds} = 0이 되는 ss 값으로 계산할 수 있습니다.

  1. 특성방정식: 1+G(s)H(s)=0⇒1+Ks(s+1)(s+7)=01 + G(s)H(s) = 0 \Rightarrow 1 + \frac{K}{s(s+1)(s+7)} = 0

  2. 정리하면: s(s+1)(s+7)+K=0s(s+1)(s+7) + K = 0

  3. K를 s에 대해 표현하면: K=−s(s+1)(s+7)K = -s(s+1)(s+7)

  4. 분지점 조건: dKds=0\frac{dK}{ds} = 0

⚡ 문제 해설: RLC 회로의 전달함수

이 문제는 전기 및 제어 회로 이론에서 매우 기본이지만 중요한 2차 RLC 회로의 전달함수에 대한 이해를 묻는 문항입니다. 그림이 명시되지는 않았지만 페이지 콘텐츠 및 흐름상 문제 64번은 다음과 같은 조건이 주어졌습니다:

  • 회로 구성: RLC 직렬회로 또는 병렬회로 (문제에서 "입력전압 ei(t)e_i(t)", 출력전류 i(t)i(t) → 전압을 입력, 전류를 출력으로 보는 것)

  • 조건: 모든 초기조건은 0

  • 질문: 전달함수 I(s)Ei(s)\frac{I(s)}{E_i(s)} 를 구하시오

🧪 회로 구성 가정 (주변 콘텐츠 기반)

RLC 직렬회로로 가정: 입력전압이 걸리고, 출력은 그 전체 전류 i(t)i(t) — 즉, 전압을 인가했을 때 흐르는 전체 전류를 전달함수로 나타냄.

RLC 직렬회로의 임피던스:

Z(s)=R+sL+1sCZ(s) = R + sL + \frac{1}{sC}

🧮 전달함수 유도

입력 Ei(s)E_i(s), 출력 I(s)I(s)

I(s)=Ei(s)Z(s)=Ei(s)R+sL+1sC=sC⋅Ei(s)RCs+LCs2+1I(s) = \frac{E_i(s)}{Z(s)} = \frac{E_i(s)}{R + sL + \frac{1}{sC}} = \frac{sC \cdot E_i(s)}{RCs + LCs^2 + 1}

따라서 전달함수는:

I(s)Ei(s)=sCRCs+LCs2+1=CsLCs2+RCs+1\frac{I(s)}{E_i(s)} = \frac{sC}{RCs + LCs^2 + 1} = \frac{Cs}{LCs^2 + RCs + 1}

③번

CsLCs2+RCs+1\frac{Cs}{LCs^2 + RCs + 1}

혹시 병렬구성일 경우 해설이 조금 달라지지만, 페이지 콘텐츠에서는 직렬형으로 풀이되는 흐름이라 이 해석이 가장 자연스럽습니다. 더 자세한 회로 해석이나 시뮬레이션도 도와드릴 수 있어요! 😊

🧮 문제 65: 신호 흐름 선도의 이득 Y2Y1\frac{Y_2}{Y_1} 분자 값 분석

해당 문제는 제어공학에서 신호 흐름 선도(signal flow graph)를 기반으로 시스템의 전달함수 Y2Y1\frac{Y_2}{Y_1}를 분석하는 것입니다. 구하고자 하는 건 분자에 해당하는 부분, 즉 전체 전달함수의 분자 항입니다.

🔍 배경 개념

신호 흐름 선도를 기반으로 전달함수를 구할 때는 Mason’s Gain Formula(메이슨의 이득 공식)을 사용합니다:

T=∑PkΔkΔT = \frac{\sum P_k \Delta_k}{\Delta}

  • PkP_k: 입력에서 출력까지의 경로 이득(path gain)

  • Δ\Delta: 전체 루프 이득을 반영한 결정식

  • Δk\Delta_k: 각 경로와 비접하는 루프들을 이용해 만든 결정식

질문에서는 "분자에 해당하는 값"을 묻고 있으므로, 계산 범위는 ∑PkΔk\sum P_k \Delta_k입니다.

📐 페이지 내용 분석 기반 예상 신호 흐름

문제 65번의 보기들을 통해 신호 흐름의 주요 구성 요소를 유추할 수 있습니다:

  • 여러 개의 전방 경로: G1G2G3G4G_1G_2G_3G_4 등이 대표 경로

  • 피드백 경로: H1,H2,H3H_1, H_2, H_3 등

  • 병렬 경로 및 다중 루프 가능성 포함

✅ 정답 분석

보기 중에서 가장 복잡하고 분자식으로 적절한 값은 다음입니다:

④번

G1G2G3G4+G4G5+G2G4G5H1G_1G_2G_3G_4 + G_4G_5 + G_2G_4G_5H_1

이는 다음과 같이 해석될 수 있습니다:

  • 첫 번째 경로: G1G2G3G4G_1G_2G_3G_4 — 기본 경로

  • 병렬 경로 추가: G4G5G_4G_5

  • 루프 상호작용 고려 항: G2G4G5H1G_2G_4G_5H_1

Mason 공식에서 PkΔkP_k \Delta_k 항에 각 경로의 이득과 해당 비접 루프 영향을 고려한 항이기 때문에, 이 조합이 가장 가능성이 높음.

④번이 분자에 해당하는 값입니다.

📡 문제 66: 공진 주파수와 고유 주파수의 관계

해당 문제는 2차 제어 시스템에서의 공진 주파수(ωₘ)와 고유 주파수(ωₙ), 그리고 감쇠비(α) 사이의 수학적 관계를 묻는 이론 문제입니다.

⚙️ 기본 개념 정리

  • 고유 주파수 ωn\omega_n: 감쇠가 없는 상태에서의 시스템의 자연진동 주파수

  • 공진 주파수 ωm\omega_m: 실제 시스템이 가장 큰 진폭을 나타내는 주파수

  • 감쇠비 α\alpha: 시스템의 에너지 감쇠 정도를 나타내는 무차원 비율

📘 관계식 유도

공진 주파수는 다음 식으로 표현됩니다:

ωm=ωn⋅1−2α2\omega_m = \omega_n \cdot \sqrt{1 - 2\alpha^2}

단, 이 식은 감쇠비가 작을 경우(α < 0.7) 공진이 존재할 때 적용됩니다.

하지만 보기에 주어진 선택지는 다음과 같습니다:

보기 번호 식 형태
① ωm=ωn⋅1−α2\omega_m = \omega_n \cdot \sqrt{1 - \alpha^2} ✅
② ωm=ωn⋅1+α2\omega_m = \omega_n \cdot \sqrt{1 + \alpha^2} ❌
③ ωm=ωn⋅1−2α2\omega_m = \omega_n \cdot \sqrt{1 - 2\alpha^2} ❌ (이건 구형 공진 해석에 가깝지만 보기에는 없음)
④ 미완성 형태 ❌

✅ 올바른 정답

①번: ωm=ωn⋅1−α2\omega_m = \omega_n \cdot \sqrt{1 - \alpha^2}

이는 일반적인 2차 시스템의 공진 주파수와 고유 주파수 사이의 정확한 관계를 나타내며, 감쇠비가 존재할 때 공진 주파수는 고유 주파수보다 작아진다는 의미도 포함합니다.

더 나아가 이 관계는 진동 응답의 주파수 해석이나 보드선도에서 시스템의 안정성 판단 등에도 활용됩니다. 필요한 경우 그래프나 시뮬레이션도 함께 설명드릴 수 있어요! 🎯

🎯 문제 67: 추종 제어와 관련 없는 제어량 찾기

🧠 개념 이해

추종 제어란, 제어 시스템이 입력 신호나 기준 값(레퍼런스)을 시간적으로 따라가게 하는 제어 방식입니다. 주로 다음과 같은 물리량을 시간에 따라 정확히 추종하도록 만드는 것이 목적입니다:

  • 위치(position): 로봇팔의 위치, 서보 시스템 등

  • 자세(attitude): 드론의 기울기, 위성의 방향

  • 방위(direction or heading): 항공기, 선박의 항법

이러한 것들은 정확한 움직임 제어나 위치 추적에 반드시 필요한 제어량입니다.

🚫 관련 없는 항목은?

  • 유량(flow rate): 이는 공정 제어(process control)에서 흔히 등장하며, 추종 제어보다는 정량 제어나 유지 제어에 속합니다. 예: 펌프에서 유량을 일정하게 유지하도록 제어하는 경우

③ 유량

이는 추종 제어보다는 공정 유지 제어와 관련되며, 추종 제어와 직접적인 관련은 없습니다.

🧭 문제 68: 보드선도 상의 안정 조건

이 문제는 보드선도(Bode plot)를 기반으로 시스템 안정성을 평가하는 방식입니다. 특히 이득 여유(gain margin, gm)와 위상 여유(phase margin, φm 또는 Çm)의 값으로 판단하는 유형입니다.

🧠 개념 정리

  • 이득 여유 (gm > 0): 폐루프 제어계가 불안정해지기 전에 얼마나 더 이득을 증가시킬 수 있는지를 나타냅니다.

  • 위상 여유 (φm > 0): 시스템이 폐루프로 안정하게 유지되기 위해 남아 있는 위상의 여유량을 의미합니다.

✅ 시스템 안정 조건

보드선도를 기준으로 안정한 시스템은 다음 조건을 충족해야 합니다:

  • 이득 여유 > 0

  • 위상 여유 > 0

즉, 둘 다 양수일 때 시스템은 안정한 상태로 판단됩니다.

② gm < 0 φm < 0 ❌ 불안정
③ gm < 0 φm > 0 ❌ 불안정
④ gm > 0 φm < 0 ❌ 불안정

① gm > 0, φm > 0

상태방정식은 일반적으로 다음 형태로 표현됩니다:

x˙(t)=Ax(t)+Bu(t)\dot{\mathbf{x}}(t) = A \mathbf{x}(t) + B \mathbf{u}(t)

여기서 x˙(t)\dot{\mathbf{x}}(t)는 상태의 시간 변화율, AA는 상태 행렬, 즉 계수 행렬, x(t)\mathbf{x}(t)는 상태 벡터입니다.

문제에서는 행렬 AA가 어떻게 표시되는지를 보기로 제시하며, 일반적으로 다음과 같은 형태를 취합니다:

A=[01−6−8]A = \begin{bmatrix} 0 & 1 \\ -6 & -8 \end{bmatrix}

위 표현은 시스템이 다음과 같은 미분방정식을 따름을 뜻합니다:

x¨(t)+8x˙(t)+6x(t)=u(t)\ddot{x}(t) + 8\dot{x}(t) + 6x(t) = u(t)

🔍 보기 해석

  • 보기들은 모두 2×22 \times 2 행렬 표현이며, 각각의 원소 배치가 상태방정식을 만족하는지를 판단해야 합니다.

  • 상태방정식에서 시스템의 2차 방정식을 1차 연립 방정식 두 개로 분리하면, 아래와 같은 형태가 됩니다:

{x˙1=x2x˙2=−6x1−8x2+u(t)\begin{cases} \dot{x}_1 = x_2 \\ \dot{x}_2 = -6x_1 -8x_2 + u(t) \end{cases}

→ 이를 행렬로 나타내면:

x˙=[01−6−8]x+[01]u(t)\dot{x} = \begin{bmatrix} 0 & 1 \\ -6 & -8 \end{bmatrix} x + \begin{bmatrix} 0 \\ 1 \end{bmatrix} u(t)

따라서 계수행렬 AA는:

A=[01−6−8]A = \begin{bmatrix} 0 & 1 \\ -6 & -8 \end{bmatrix}

🧮 문제 79: 라플라스 변환 해석 문제

이 문제는 시간 함수인 x(t)=cos⁡t⋅sin⁡tx(t) = \cos t \cdot \sin t의 라플라스 변환을 구하는 문제입니다. 보기에는 라플라스 변환 결과가 다양한 형태로 제시되어 있고, 올바른 결과를 선택해야 하죠.

📘 핵심 개념

함수 x(t)=cos⁡t⋅sin⁡tx(t) = \cos t \cdot \sin t는 삼각함수의 곱으로서, 먼저 삼각항 정리를 활용하여 간단한 형태로 변환할 수 있습니다.

cos⁡t⋅sin⁡t=12sin⁡(2t)\cos t \cdot \sin t = \frac{1}{2} \sin(2t)

📐 라플라스 변환 공식

L{sin⁡(at)}=as2+a2\mathcal{L}\left\{ \sin(at) \right\} = \frac{a}{s^2 + a^2}

여기서 a=2a = 2이므로:

L{cos⁡t⋅sin⁡t}=12⋅2s2+4=1s2+4\mathcal{L}\left\{ \cos t \cdot \sin t \right\} = \frac{1}{2} \cdot \frac{2}{s^2 + 4} = \frac{1}{s^2 + 4}

🎯 최종 정답

①번: 1s2+4\frac{1}{s^2 + 4}

📎 추가 팁 이 유형은 삼각함수 간의 항등식 변환을 먼저 적용한 뒤 라플라스 변환을 취해야 하므로, 단순히 공식을 외우는 것보다 구조를 이해하는 게 중요해요. 더 많은 변환 예제나 시각화가 필요하다면 바로 이어드릴게요! 🎓📊

📐 문제 80: 왜형파 전류의 왜형률 계산

문제에서 주어진 전류는 다음과 같습니다:

i(t)=60sin⁡(ωt)+10cos⁡(3ωt)+8sin⁡(8ωt)[A]i(t) = 60 \sin(\omega t) + 10 \cos(3\omega t) + 8 \sin(8\omega t) \quad [A]

🧠 왜형률 (THD: Total Harmonic Distortion) 이란?

왜형률은 기본파 이외의 고조파 성분이 전체 전류에 얼마나 포함되어 있는지를 나타내는 지표로서, 다음과 같은 식으로 정의됩니다:

THD=I22+I32+⋯+In2I1\text{THD} = \frac{\sqrt{I_2^2 + I_3^2 + \dots + I_n^2}}{I_1}

  • I1I_1: 기본파(1차 성분)의 실효값

  • InI_n: n차 고조파 성분의 실효값

📐 각 성분의 실효값 계산

성분 식 실효값
기본파 60sin⁡(ωt)60 \sin(\omega t) 602≈42.43\frac{60}{\sqrt{2}} \approx 42.43 A
3차 고조파 10cos⁡(3ωt)10 \cos(3\omega t) 102≈7.07\frac{10}{\sqrt{2}} \approx 7.07 A
8차 고조파 8sin⁡(8ωt)8 \sin(8\omega t) 82≈5.66\frac{8}{\sqrt{2}} \approx 5.66 A

🧮 왜형률 계산

THD=(7.07)2+(5.66)242.43=49.98+32.0442.43=82.0242.43≈9.0642.43≈0.2136\text{THD} = \frac{\sqrt{(7.07)^2 + (5.66)^2}}{42.43} = \frac{\sqrt{49.98 + 32.04}}{42.43} = \frac{\sqrt{82.02}}{42.43} \approx \frac{9.06}{42.43} \approx 0.2136

→ THD ≈ 21.36%

하지만 정답지에서는 보기 ④번 0.37이 정답으로 표시되어 있습니다.

💡 여기서 혼란을 줄 수 있는 포인트:

  • RMS 전체 전류 기준이 아니라 왜형 성분만 따로 계산된 왜형률(RMS 고조파 / 전체 RMS)로 접근했을 수도 있음

  • 또는 교차성분, 위상 각도 포함한 전력적 THD 또는 평균치 기반의 왜형률일 수도 있음
