문제

61. 의 역 z 변환은?(정답률:67%)      
     1. 1 – e-at
     2. 1 + e-at
     3. t • e-at
     4. t • eat

     정답 : [1]

해설

물론입니다, Jong! 이 문제는 역 Z 변환(inverse Z-transform)에 관한 것입니다. Z 변환은 이산신호를 복잡한 주파수 영역으로 변환하여 시스템 분석을 가능하게 하는 도구이며, 그 반대로 신호를 시간 영역으로 되돌리는 것이 역 Z 변환입니다.

🧩 문제 이해

질문: 다음 표현의 역 Z 변환은?

math
F(z) = \frac{z}{z-1} - \frac{z}{z - e^{-aT}}

이 표현은 두 개의 표준 Z 변환을 빼는 구조입니다.

📘 기본 Z 변환 공식

시간 영역 함수Z 영역 변환설명
f(t)=u(t)f(t) = u(t)zz−1\frac{z}{z - 1}단위 계단 함수
f(t)=e−aTtu(t)f(t) = e^{-aTt}u(t)zz−e−aT\frac{z}{z - e^{-aT}}지수 감쇠 함수

🔁 Z 변환의 역 변환

이제 아래 식을 분석해보면:

math
F(z) = \frac{z}{z - 1} - \frac{z}{z - e^{-aT}}

이 Z 변환은 시간 영역에서 다음 두 함수의 차로 해석됩니다:

math
f(t) = u(t) - e^{-aTt}u(t)

즉, 단위 계단 함수에서 감쇠 지수 함수를 뺀 것.

이것은 수학적으로 다음과 같이 정리됩니다:

math
f(t) = (1 - e^{-at})u(t)

따라서 정답은 1번 1−e−at1 - e^{-at} 이며, 이는 시간에 따라 지수적으로 증가하는 함수에 기반한 출력의 차이를 나타냅니다.

📝 요약

  • Z 영역에서 두 개의 함수 차이를 통해 시간 영역의 출력을 찾는다.

  • 역 Z 변환 후 결과는 f(t)=1−e−atf(t) = 1 - e^{-at}

  • 선택지 중 정답은 1번

문제

62. 다음의 특성 방정식 중 안정한 제어시스템은?(정답률:77%)      
     1. s3+3s2+4s+5=0
     2. s4+3s3-s2+s+10=0
     3. s5+s3+2s2+4s+3=0
     4. s4-2s3-3s2+4s+5=0

     정답 : [1]

해설

물론입니다! 이 문제는 제어 시스템의 안정성 판단을 위한 특성 방정식에 대한 것입니다. 특성 방정식의 해, 즉 극점(pole)이 모두 복소평면의 좌측반평면(LHP)에 있을 경우 해당 시스템은 안정하다고 판단합니다.

🎯 문제 핵심

다음 중 안정한 제어 시스템의 특성 방정식을 고르시오:

보기특성 방정식
1번s3+3s2+4s+5=0s^3 + 3s^2 + 4s + 5 = 0
2번s4+3s3−s2+s+10=0s^4 + 3s^3 - s^2 + s + 10 = 0
3번s5+s3+2s2+4s+3=0s^5 + s^3 + 2s^2 + 4s + 3 = 0
4번s4−2s3−3s2+4s+5=0s^4 - 2s^3 - 3s^2 + 4s + 5 = 0

🧪 안정성 판단 기준

  • 특성 방정식의 모든 계수가 양수여야 합니다. (Routh-Hurwitz 기준의 전제 조건)

  • 차수가 연속적이고 누락된 항이 없어야 합니다.

  • 계수에 부호 변화가 없어야 합니다.

🔍 보기별 분석

  1. s3+3s2+4s+5s^3 + 3s^2 + 4s + 5

    • 모든 항이 양의 계수.

    • 연속적인 차수 → s3,s2,s1,s0s^3, s^2, s^1, s^0

    • 부호 변화 없음 → ✅ 안정 조건 만족

  2. s4+3s3−s2+s+10s^4 + 3s^3 - s^2 + s + 10

    • −s2-s^2 → 부호 변화 있음 ❌

    • 일부 계수는 음수 → 불안정 가능성 있음

  3. s5+s3+2s2+4s+3s^5 + s^3 + 2s^2 + 4s + 3

    • s4s^4 항 누락 ❌ → 차수 연속성 깨짐

  4. s4−2s3−3s2+4s+5s^4 - 2s^3 - 3s^2 + 4s + 5

    • s4,s3,s2s^4, s^3, s^2에 음수 계수 포함 → 부호 변화 다수 ❌

✅ 최종 정답: 1번

정답인 1번만이 모든 계수가 양수이고, 차수도 연속적이며, 부호 변화가 없는 안정적인 시스템의 특성 방정식입니다.

문제

63. 그림의 신호흐름선도에서 전달함수 는?(정답률:57%)      
     1.
     2.
     3.
     4.

     정답 : [4]

해설

좋습니다, Jong! 이 문제는 신호흐름선도(Signal Flow Graph)를 기반으로 시스템의 전달함수(Transfer Function)를 계산하는 내용입니다. 전달함수는 보통 Mason의 이득 공식(Mason's Gain Formula)을 이용해 계산합니다.

🧠 Mason의 이득 공식

전달함수 TT 는 다음과 같이 계산됩니다:

T=∑i=1NPiΔiΔT = \frac{\sum_{i=1}^{N} P_i \Delta_i}{\Delta}

  • PiP_i: 입력에서 출력까지의 i번째 경로의 이득

  • Δ\Delta: 전체 시스템의 루프 감쇠식

  • Δi\Delta_i: 루프를 고려한 i번째 경로에 영향을 주지 않는 루프들의 감쇠

📐 해석 요약

해설에 나온 공식:

aaa / [1 − (ab + ab + ab − abab − abab)]

이를 구조적으로 해석하면:

  • aaa: 전체 입력부터 출력까지 경로의 이득 P=aaaP = aaa

  • ab + ab + ab: 3개의 루프 존재, 각각 a→b로 구성된 루프

  • abab, abab: 비상호 독립인 두 루프의 곱(비독립 루프 페어의 영향을 뺌)

전체 감쇠식 Δ\Delta는:

Δ=1−(ab+ab+ab)+(abab+abab)\Delta = 1 - (ab + ab + ab) + (abab + abab)

✅ 결론

전달함수:

T(s)=aaa1−(ab+ab+ab−abab−abab)T(s) = \frac{aaa}{1 - (ab + ab + ab - abab - abab)}

→ 이는 보기 4번에 해당하며, 따라서 정답은 4번입니다.

📝 오류 신고 논의 요약

  • 2번이 정답이라는 애듀윌 교재의 주장은 해당 그림이 3개 블록이 병렬로 전개되었을 때 적용되는 것으로 보입니다.

  • 하지만 이 문제의 그림에서는 a→b 블록이 겹쳐서 흐르지 않음 → 루프 형태가 다르고 흐름이 분리되어 있어 해석이 달라집니다.

  • 단순히 수식 외우는 것이 아니라 선도 구조에서 신호 흐름을 정확히 파악하는 게 핵심이라는 좋은 반론이 포함되어 있습니다.

필요하시면 Mason의 공식 문제 더 풀어볼 수 있게 예제도 드릴게요! 🤖📘

문제

64. 그림과 같은 블록선도의 제어시스템에 단위계단 함수가 입력되었을 때 정상상태 오차가 0.01이 되는 a의 값은?(정답률:63%)      
     1. 0.2
     2. 0.6
     3. 0.8
     4. 1.0

     정답 : [1]

해설

물론이죠, Jong! 이 문제는 정상상태 오차(Steady-State Error)와 관련된 제어 시스템의 분석 문제입니다. 입력이 단위 계단 함수(step input)이고, 원하는 정상상태 오차가 0.01일 때, 시스템 파라미터인 a의 값을 찾는 것이 목표입니다.

🎯 문제 요약

  • 입력: 단위 계단 함수 → R(s)=1sR(s) = \frac{1}{s}

  • 전달함수: G(s)=19.8s+aG(s) = \frac{19.8}{s + a}

  • 정상상태 오차: ess=lim⁡s→0sE(s)e_{ss} = \lim_{s \to 0} sE(s)

  • 정답률: 63%, 정답: 1번 (a = 0.2)

📘 해결 과정

✅ 정상상태 오차 공식 (계단 입력일 때):

ess=lim⁡s→011+G(s)e_{ss} = \lim_{s \to 0} \frac{1}{1 + G(s)}

G(s) → 19.8s+a\frac{19.8}{s + a} 이므로:

Kp=lim⁡s→019.8s+a=19.8aK_p = \lim_{s \to 0} \frac{19.8}{s + a} = \frac{19.8}{a}

따라서:

ess=11+Kp=11+19.8a=1a+19.8a=aa+19.8e_{ss} = \frac{1}{1 + K_p} = \frac{1}{1 + \frac{19.8}{a}} = \frac{1}{\frac{a + 19.8}{a}} = \frac{a}{a + 19.8}

정상상태 오차 ess=0.01e_{ss} = 0.01일 때:

aa+19.8=0.01⇒a=0.01(a+19.8)⇒a=0.01a+0.198⇒0.99a=0.198⇒a=0.1980.99=0.2\frac{a}{a + 19.8} = 0.01 \\ \Rightarrow a = 0.01(a + 19.8) \\ \Rightarrow a = 0.01a + 0.198 \\ \Rightarrow 0.99a = 0.198 \\ \Rightarrow a = \frac{0.198}{0.99} = 0.2

✅ 최종 정답: 1번 (a = 0.2)

a 값이 작을수록 계단 입력에 대한 정상상태 오차가 줄어들며, 요구된 오차인 0.01에 정확히 맞추기 위해서는 a = 0.2가 이상적입니다.

문제

65. 그림과 같은 보드선도의 이득선도를 갖는 제어시스템의 전달함수는?(문제 오류로 가답안 발표시 2번으로 발표되었지만 확정답안 발표시 모두 정답처리 되었습니다. 여기서는 가답안인 2번을 누르면 정답 처리 됩니다.)(정답률:71%)      
     1.
     2.
     3.
     4.

     정답 : [2]

해설

좋습니다, Jong! 이 문제는 보드선도(Bode Plot)를 보고 전달함수를 추정하는 유형입니다. 전달함수의 형태는 이득(Gain)과 주파수(w)를 기반으로 하는 꺾임(Break Frequency)과 기울기(Slope)로 유추할 수 있습니다.

🔍 보드선도 해석 요약

보드선도에서 관찰할 수 있는 점은 다음과 같습니다:

  • 꺾이는 주파수:

    • 두 번 꺾이는 위치가 0.1 rad/s, 1 rad/s → 이 두 지점은 전달함수의 극점 또는 영점의 위치를 나타냅니다.

  • 최대 이득(Gain):

    • 주파수 → 0 에서 Gain = 20 dB → 이득을 로그 스케일에서 환산하면: $$ 20 = 20 \log_{10} K \Rightarrow K = 10 $$ 즉, 전달함수 상수 이득은 10임

🧠 전달함수 유추

전달함수의 일반적 형태는:

math
G(s) = K \cdot \frac{(s/z_1)(s/z_2)\dots}{(s/p_1)(s/p_2)\dots}

그래프 꺾이는 점이 0.1, 1 → 이 두 값이 분모의 극점(pole)에 해당

따라서 전달함수는 다음과 같은 구조일 것으로 추정됩니다:

math
G(s) = \frac{10}{(s + 0.1)(s + 1)}

✅ 선택지와 정답

보기에 실제 표현이 없는 관계로, 문제에서 2번을 누르면 정답 처리되고, 해설에서도 이 구조가 2번에 해당된다고 명시되어 있습니다.

따라서:

  • 꺾이는 주파수 → 두 개의 극점

  • 최대 이득 → 상수값 10

  • → 전달함수는 G(s) = 10 / (s + 0.1)(s + 1)

📌 정답: 2번

문제

66. 그림과 같은 블록선도의 전달함수 는?(정답률:67%)      
     1.
     2.
     3.
     4.

     정답 : [4]

해설

물론이죠, Jong! 이 문제는 블록선도(Block Diagram)를 분석해 제어시스템의 전달함수(Transfer Function)를 구하는 유형입니다.

🔍 핵심 개념 정리

제어 시스템에서 전달함수는 입력과 출력의 비율을 표현하는 수학적 함수로, 시스템의 동적 특성을 나타냅니다. 기본적인 폐루프(Closed-loop) 시스템에서 전달함수는 다음과 같이 정의됩니다:

T(s)=G(s)1+G(s)H(s)T(s) = \frac{G(s)}{1 + G(s)H(s)}

여기서:

  • G(s)G(s): 순방향 이득 (forward gain)

  • H(s)H(s): 되먹임 이득 (feedback gain)

🔗 문제 해설 요약

해설에 따르면, 되먹임 경로의 전달함수가 H=H1+H2H = H_1 + H_2로 주어져 있습니다. 이는 병렬 연결된 두 개의 되먹임 요소의 합으로 계산된 것입니다.

따라서 전체 시스템의 전달함수는:

T(s)=G(s)1+G(s)(H1+H2)T(s) = \frac{G(s)}{1 + G(s)(H_1 + H_2)}

이 구조에 해당하는 보기 번호는 4번, 그리고 정답률도 67%로 상당히 높은 편이네요.

✅ 결론

  • 전달함수 형태: T(s)=G(s)1+G(s)(H1+H2)T(s) = \frac{G(s)}{1 + G(s)(H_1 + H_2)}

  • 정답: 4번

67. 그림과 같은 논리회로와 등가인 것은?(정답률:64%)      
     1.
     2.
     3.
     4.

     정답 : [2]

해설

물론이죠, Jong! 이 문제는 논리회로의 등가 표현을 추리하는 문제입니다. 주어진 논리회로에서 구성된 게이트들을 조합하여 최종 출력이 어떤 논리식으로 나타나는지를 분석하는 것이 핵심입니다.

🧠 주요 포인트

논리식:

AB′ + A′B + AB

이를 하나씩 살펴보면:

  • AB′ → A=1이고 B=0일 때만 1

  • A′B → A=0이고 B=1일 때만 1

  • AB → A=1이고 B=1일 때만 1

→ 즉, 세 항의 OR(합)은 입력 (A, B) 조합 중 세 경우에만 참(True)

📊 진리표로 확인

ABAB′A′BAB결과
000000
010101
101001
110011

따라서 출력은 A=0, B=0일 때만 0, 나머지는 모두 1 → 이는 OR 게이트 출력 A+BA + B과 동일합니다.

🔁 논리식 단순화 과정

  1. AB′+A′B+ABAB′ + A′B + AB

  2. AB′+(A′+A)BAB′ + (A′ + A)B → A′+A=1A′ + A = 1, 그러므로:

  3. AB′+BAB′ + B

  4. AB′+B(1+A)=AB′+B+AB=A(B′+B)+B=A+BAB′ + B(1+A) = AB′ + B + AB = A(B′ + B) + B = A + B

✅ 최종 결론

  • 주어진 복잡한 논리식은 결국 A + B로 단순화됩니다

  • 보기 중 해당 논리식을 만족하는 회로는 2번

  • 따라서 정답: 2번

문제

68. 다음의 개루프 전달함수에 대한 근궤적의 점근선이 실수축과 만나는 교차점은?(정답률:60%)      
     1.
     2.
     3.
     4.

     정답 : [4]

해설

물론이죠, Jong! 이 문제는 근궤적(root locus)의 점근선(asymptote)에 대한 것입니다. 근궤적에서 점근선은 폐루프 극점이 무한대로 갈 때의 궤적 방향을 나타내며, 실수축과의 교차점은 점근선들이 모이는 중심(Centroid)을 의미합니다.

🔍 문제 핵심 요약

질문: 주어진 개루프 전달함수에 대한 근궤적 점근선이 실수축과 만나는 지점은?

⚙️ 공식: 점근선의 교차점(Centroid)

점근선의 교차점은 다음 공식으로 계산됩니다:

σa=∑극점−∑영점n−m\sigma_a = \frac{\sum \text{극점} - \sum \text{영점}}{n - m}

  • ∑극점\sum \text{극점}: 모든 극점의 위치 합

  • ∑영점\sum \text{영점}: 모든 영점의 위치 합

  • nn: 극점 개수

🎯 핵심 개념: 무왜형 조건이란?

통신 회선이나 고속 전송선로에서 신호가 왜곡 없이 전달되기 위해 만족해야 하는 조건으로:

RC=LGRC = LG

  • RR: 저항 (resistance per unit length)

  • LL: 인덕턴스 (inductance per unit length)

  • GG: 누설컨덕턴스 (leakage conductance per unit length)

  • CC: 커패시턴스 (capacitance per unit length)

🧮 계산 과정

문제에서 주어진 값:

  • R=100 Ω/mR = 100 \ \Omega/m

  • L=200 mH/m=0.2 H/mL = 200 \ \text{mH}/m = 0.2 \ \text{H}/m

  • G=0.5 ℧/mG = 0.5 \ \text{℧}/m

무왜형 조건 적용:

C=L⋅GR=0.2⋅0.5100=0.1100=0.001 F/mC = \frac{L \cdot G}{R} = \frac{0.2 \cdot 0.5}{100} = \frac{0.1}{100} = 0.001 \ \text{F}/m

→ 단위 변환:

0.001 F=1000 μF0.001 \ \text{F} = 1000 \ \mu \text{F}
