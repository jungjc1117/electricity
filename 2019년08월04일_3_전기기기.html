     3. 코일 상호간 큰 전자력에 의한 코일 파손
     4. 큰 단락전류 후 점차 감소하여 지속 단락전류 유지

해설

다음은 주어진 텍스트와 주변 페이지 콘텐츠를 바탕으로 설명한 내용입니다:

🔍 문제 개요

문제 41: “동기발전기의 돌발 단락 시 발생되는 현상으로 틀린 것은?”

선택지:

  1. 큰 과도전류가 흘러 권선 소손

  2. 단락전류는 전기자 저항으로 제한

  3. 코일 상호간 큰 전자력에 의한 코일 파손

  4. 큰 단락전류 후 점차 감소하여 지속 단락전류 유지

✅ 정답: 2번 (틀린 설명) 📊 정답률: 53%

📘 개념 설명

동기발전기의 돌발 단락(short-circuit) 상황은 회로의 갑작스러운 이상으로 인해 고장 전류가 급격히 흐르는 상황을 말합니다. 이때 발생하는 주요 현상은 다음과 같습니다:

  • 🔥 큰 과도전류(Transient Current): 단락 직후에는 매우 큰 전류가 발생하여 기기의 권선이 손상될 수 있음 → 1번: 맞는 설명

  • ⚡ 코일 상호 간 전자력: 강한 전류로 인해 권선 간에 큰 전자력이 작용, 기계적 파손 가능 → 3번: 맞는 설명

  • 📉 점차 감소하는 전류: 초기 과도전류 이후 점차 정상 단락 전류로 감소 → 4번: 맞는 설명

  • ❌ 전기자 저항으로 제한된다: 일반적으로 동기기의 단락전류는 전기자 저항이 아닌 전기자 리액턴스, 특히 누설 리액턴스(leakage reactance) 에 의해 제한됩니다 → 2번: 틀린 설명 (정답)

💡 참고 해설 요약

  • 전기자 저항은 매우 작기 때문에 단락전류 제한 역할이 거의 없음

  • 실질적인 제한은 전기자 리액턴스 (Xₐ)와 누설 리액턴스에 의해 이루어짐

  • 해설에서 말하는 “코일 = 한류 리액터”는 단락 시 전류를 제한하는 요소로 이해하면 됨

  2. 열용량이 적어 고온에 약하다

⚡ SCR의 주요 특징

SCR(Silicon Controlled Rectifier), 또는 사이리스터는 반도체 스위칭 소자로서 고전력 제어에 널리 쓰입니다. 주요 특징은 다음과 같습니다:

  • 🧨 과전압에 약함: 구조적으로 PNPN 형식이며, 높은 전압에 취약 → 1번: 맞는 설명

  • 🔥 열용량이 낮고 고온에 취약: 고전력 취급 시 내부 발열 많음, 과열되면 동작 불안정 → 2번: 맞는 설명

  • ⚡ 양극 전압강하: 도통 상태일 때 문턱 전압(약 0.7V~1.5V) 정도만 발생하며 매우 작음 → 3번: 틀린 설명 (정답)

  • ⏱️ 도통까지의 응답 시간 빠름: 게이트 전류 인가 시 매우 짧은 시간에 도통 → 4번: 맞는 설명

📘 해설 요약

  • 사이리스터는 도통 시 거의 저항이 없는 상태로 전류가 흐르며, 양극 전압강하는 작다.

  • 보통 문턱 전압 약 0.7V 수준이며, 큰 전압강하는 발생하지 않는다.

  • 따라서 “전류가 흐를 때 양극 전압강하가 크다”는 설명은 틀림.

🌀 문제 개요

문제 43: “터빈 발전기의 냉각을 수소냉각방식으로 하는 이유로 틀린 것은?”

🌬️ 수소냉각 방식의 특징과 장점

수소(H₂) 냉각은 대형 터빈 발전기의 효율적인 냉각 방식으로 널리 사용됩니다. 그 이유는 다음과 같습니다:

  • 🔻 풍손(환기 손실) 감소: 수소는 공기보다 밀도와 점성이 낮아 풍손이 약 1/10로 줄어듭니다 → 1번: 맞는 설명

  • 💨 열전도율 우수: 수소는 열전도율이 높아 냉각 효율이 좋고, 필요한 냉각 장치의 크기를 줄일 수 있음 → 2번: 맞는 설명

  • 🧪 산화 억제: 산소 대신 수소를 사용하는 환경에서는 절연물의 산화가 억제되어 절연 열화가 적고 수명이 길어짐 → 3번: 맞는 설명

🚫 4번 선택지 왜 틀렸을까?

  • 수소냉각방식에서는 압력용기 구조를 사용하며 밀폐형 설계가 필수입니다.

  • 만약 반폐형 구조로 하면 수소가 외부의 산소와 접촉하여 폭발 위험이 발생합니다.

  • 따라서 "반폐형 구조이기 때문에 이물질 침입이 없다"는 설명은 물리적으로 불가능하고 안전상 치명적인 오류입니다.

→ 4번은 틀린 설명으로 정답입니다.

🧭 문제 개요

문제 44: "단상 유도전동기의 특징을 설명한 것으로 옳은 것은?"

선택지:

  1. 기동 토크가 없으므로 기동장치가 필요하다

  2. 기계손이 있어도 무부하 속도는 동기속도보다 크다

  3. 권선형은 비례추이가 불가능하며, 최대 토크는 불변이다

  4. 슬립은 0＞S＞-1이고, 2보다 작고 0이 되기 전에 토크가 0이 된다

✅ 정답: 1번 📊 정답률: 60%

⚙️ 단상 유도전동기의 특징

단상 유도전동기(Single-phase Induction Motor)는 구조적으로 회전 자계(rotating magnetic field)가 자체적으로 발생하지 않기 때문에, 기동 시 토크가 존재하지 않습니다. 따라서 다음과 같은 특징이 있습니다:

  • 🔌 기동 토크 없음: 자체 기동이 불가능하여 기동장치(예: 기동 콘덴서, 보조권선 등)가 필요 → 1번은 정확한 설명

  • ❌ 2번 설명 오류: 무부하 속도는 동기속도보다 항상 작다. 기계 손실이 있더라도 이를 초과할 수는 없음

  • ❌ 3번 설명 오류: 권선형은 비례추이 가능 (속도 제어가 용이)하며 최대 토크는 부하 조건에 따라 달라짐

  • ❌ 4번 설명 오류: 슬립(S)은 0 < S < 1의 범위를 갖는 것이 일반적이며, 0보다 작거나 2보다 작다는 표현은 명확히 잘못됨

💡 추가 해설

  • 단상 유도전동기는 회전 자계를 만들기 위해 페이저를 나누거나 기동권선과 보조콘덴서를 사용

  • 대표적인 종류로는 분상기동형, 콘덴서 기동형, 콘덴서 운전형, 셰이딩코일형 등이 있음

  • 일반적으로 소형 가전기기, 팬, 냉장고 등에 쓰이며, 구조가 단순하고 경제적임

  • ⚡ 절연레벨은 유입변압기보다 낮다 → 유입변압기는 절연유로 높은 절연 성능을 갖추므로 같은 정격 전압 기준에서는 유입식이 몰드보다 절연레벨이 더 높음 → 4번: 맞는 설명처럼 보이지만 문제의 의도상 ‘틀린 것으로 간주’

⚙️ 문제 개요

문제 46: “유도전동기의 회전속도를 NN(rpm), 동기속도를 NsN_s(rpm)이라 하고, 순방향 회전자계의 슬립을 ss라고 하면, 역방향 회전자계에 대한 회전자 슬립은?”

선택지:

  1. s−1s - 1

  2. 1−s1 - s

  3. s−2s - 2

  4. 2−s2 - s ✅ 정답

📊 정답률: 66%

🌀 슬립(Slip)의 개념

슬립은 유도전동기에서 회전자와 회전자계(자속)의 속도 차이를 나타내는 비율입니다:

  • 순방향 슬립: $$ s = \frac{N_s - N}{N_s} $$

  • 여기서 NsN_s: 동기속도 NN: 회전자속도

🔁 역방향 회전자계에 대한 슬립 계산

단상 유도전동기에서는 회전자계가 순방향과 역방향으로 동시에 존재합니다. 이를 분석하면 다음과 같은 결과가 도출됩니다:

  • 정상분(순방향) 회전자계의 주파수: (1−s)f(1 - s)f

  • 역상분(역방향) 회전자계의 주파수: f+(1−s)f=(2−s)ff + (1 - s)f = (2 - s)f

  • 단상 유도전동기는 회전 자계를 형성하기 위해 비대칭 자속을 발생시키는데, 이로 인해 정상분과 역상분 자속이 생깁니다.

⚙️ 병렬운전 조건과 부하분담 원리

단상 변압기를 병렬로 운전하는 경우, 안정성과 효율성을 위해 다음 조건을 만족해야 합니다:

✅ 병렬운전 기본 조건

  • 정격전압 동일

  • 극성 동일

  • %임피던스(임피던스 전압) 적절한 관계 유지

  • 전압비 동일

📐 부하분담 공식

병렬운전 시 각 변압기에 분배되는 전류는 그 변압기의 임피던스 크기에 반비례합니다.

PBPA=IB⋅%ZAIA⋅%ZB\frac{P_B}{P_A} = \frac{I_B \cdot \%Z_A}{I_A \cdot \%Z_B}

  • 전류는 용량에 비례: IA∝PAI_A \propto P_A, IB∝PBI_B \propto P_B

  • 대입하면 다음 관계 도출:

PBPA=PB⋅%ZAPA⋅%ZB⇒%ZA%ZB=PAPB\frac{P_B}{P_A} = \frac{P_B \cdot \%Z_A}{P_A \cdot \%Z_B} \Rightarrow \frac{\%Z_A}{\%Z_B} = \frac{P_A}{P_B}

→ 따라서 %임피던스는 변압기 용량에 반비례해야 적절한 부하분담이 이루어집니다.

🔄 전력변환기기란?

전력변환기기(Power Converter)는 전기의 형태나 특성(전압, 전류, 주파수, 위상 등)을 다른 형태로 변환하는 기기를 말합니다. 이를테면:

  • 컨버터 (Converter): AC ↔ DC 변환 (통칭적 용어로 사용됨)

  • 정류기 (Rectifier): AC → DC 변환

  • 인버터 (Inverter): DC → AC 변환

이들 기기는 주로 전력전자 기술을 기반으로 작동하며, 각종 산업 기기와 에너지 시스템에서 핵심적 역할을 합니다.

❌ 유도전동기는 왜 틀린 선택인가?

  • 유도전동기(Induction Motor)는 전력변환기기가 아닌 전기기기(Electrical Machine)입니다.

  • 전력을 기계적인 회전력으로 "사용"하는 기기이지, 전력의 형태를 **"변환"하는 기기는 아닙니다. → 즉, 출력 기기이지 변환 기기가 아님.

📝 한줄 요약

  • 컨버터, 정류기, 인버터는 모두 전력을 변환하는 장치 ✅

  • 유도전동기는 전력 변환 장치가 아닌 기계적 동력 생성 장치 ❌ → 정답!

🔍 핵심 개념 정리

농형 유도전동기(Squirrel-cage Induction Motor)는 2차측(회전자)에 권선을 연결하지 않고 알루미늄 또는 구리 막대로 회전자 도체를 구성한 구조입니다. 이러한 구조는 속도제어가 제한적이며, 사용 가능한 제어법이 다음과 같이 구분됩니다:

속도제어법 적용 가능 유도전동기 제어 방법 설명
극수 변환법 ✅ 농형 유도전동기 동기속도 Ns = 120f/P에서 P(극수)를 바꿔 속도 제어
종속 접속법 ❌ 농형 불가 / 권선형 전용 2차 권선 연결 조합 변경으로 속도 제어
2차 저항제어법 ❌ 농형 불가 / 권선형 전용 회전자 회로에 저항 삽입하여 슬립 조절
2차 여자제어법 ❌ 농형 불가 / 권선형 전용 회전자 여자 방식 조절로 회전속도 변경

⚡ 왜 ‘극수 변환법’이 맞는가?

  • 농형 유도전동기는 회전자 구조상 외부 회로 제어 불가능

  • 2차측 접근이 제한적이므로 2차 저항 제어나 종속 접속법은 적용 불가

  • 극수(P)를 변경하면 동기속도가 달라지므로 속도 제어 가능

예를 들어,

  • 극수가 4 → 동기속도 1800 rpm (60 Hz 기준)

  • 극수가 6 → 동기속도 1200 rpm

→ 단순하면서도 농형에 효과적인 방법입니다.

궁금하시면 극수 변환법을 쓰는 대표적 용도 예시도 드릴게요 — 예를 들어 엘리베이터, 송풍기 등에 많이 쓰인답니다 😊

문제 53: 정격전압이 100V, 정격전류가 50A인 분권발전기의 유기기전력(Generated EMF)을 구하는 문제입니다. 주어진 조건:

🔍 개념 정리: 유기기전력 계산 방법

분권발전기에서 유기기전력 EE은 전기자 내부에서 발생한 전압을 의미하며, 단자전압(V)과 전기자 내부의 전압 강하(Ia·Ra)를 합산하여 계산합니다.

E=V+Ia⋅RaE = V + I_a \cdot R_a

여기서:

  • VV: 출력 단자전압

  • IaI_a: 전기자 전류 (이 문제에서는 전부가 전기자전류로 가정됨)

  • RaR_a: 전기자 저항

🧮 계산 과정

E=100 V+(50 A×0.2 Ω)=100 V+10 V=110 VE = 100\,V + (50\,A \times 0.2\,\Omega) = 100\,V + 10\,V = 110\,V

→ 따라서 유기기전력은 110V이며, 정답은 1번입니다.

📘 참고 개념

  • 전기자 반작용은 자속을 감소시켜 유기기전력을 저하시킬 수 있으나, 무시하므로 단순 계산이 가능합니다.

  • 백분율 저항강하 %R=3%\%R = 3\%

⚙️ 전압변동률 공식

변압기의 전압변동률(%)은 부하의 성격에 따라 다음과 같은 공식이 사용됩니다:

전압변동률=%R⋅cos⁡θ±%X⋅sin⁡θ\text{전압변동률} = \%R \cdot \cos\theta \pm \%X \cdot \sin\theta

  • 역률이 진상(앞선)이면: − 사용

🧮 계산 과정

  • cos⁡θ=0.8\cos\theta = 0.8, sin⁡θ=0.6\sin\theta = 0.6 (역률 80% 기준)

  • 백분율 저항강하: 3%×0.8=2.4%3\% \times 0.8 = 2.4\%

  • 백분율 리액턴스강하: 4%×0.6=2.4%4\% \times 0.6 = 2.4\%

전압변동률=2.4%+2.4%=4.8%\text{전압변동률} = 2.4\% + 2.4\% = 4.8\%

→ 따라서 정답은 [3]번: 4.8%

  • 변압기의 내부 임피던스 구성(저항 + 리액턴스)에 따라 부하 조건이 달라지면 전압 강하가 달라지므로 역률 반영은 매우 중요합니다.

다른 유형의 부하 — 진상 역률이나 동기 조상기 연결 시 전압변동률이 어떻게 달라지는지도 함께 살펴보면 이해가 더 깊어질 거예요 😊

🔄 문제 개요

문제: 정류자형 주파수변환기의 회전자에 주파수 f1f_1의 교류를 가했을 때, 브러시 사이에 나타나는 주파수 fcf_c를 설명한 것 중 틀린 것은?

⚙️ 개념 정리

정류자형 주파수변환기(Commutator-type Frequency Converter)는 회전기 방식으로, 유도전동기의 원리와 매우 유사한 기계입니다. 그 작동 메커니즘은 다음과 같습니다:

  • 슬립(s):

s=Ns−NNss = \frac{N_s - N}{N_s}

단, 정방향 회전 시

  • 브러시 사이 전압의 주파수 fcf_c는 슬립과 공급 주파수에 따라 다음과 같이 결정됩니다:

fc=s⋅f1f_c = s \cdot f_1

🔍 선택지 분석

번호 설명 내용 옳고 그름 해설
1 회전자 정지 → s=1s = 1 → fc=f1f_c = f_1 ✅ 옳음 정지 시 슬립이 1
2 반시계방향 n=nsn = n_s → s=0s = 0 → fc=0f_c = 0 ✅ 옳음 완전 동기 회전
3 반시계방향 n<nsn < n_s → 0<s<10 < s < 1 → fc=s⋅f1f_c = s \cdot f_1 ✅ 옳음 표준 회전자 상태
4 시계방향 역회전 시 s>1s > 1 → fc>f1f_c > f_1 → ❌ 틀림 역회전의 경우 슬립이 1보다 크기 때문에 브러시 사이 주파수는 오히려 증가함

→ 즉, 4번은 틀린 설명이며 정답입니다.

🧠 덧붙이는 이해 포인트

  • 정류자형 변환기는 주파수 변환뿐만 아니라 속도 제어와 관련된 특수 응용에도 사용됩니다.

  • 역회전 시 슬립 정의가 바뀌며 s=Ns+NNs>1s = \frac{N_s + N}{N_s} > 1 가 되어 주파수가 커지게 됩니다.

필요하시면 유도기 슬립과 주파수 관계에 대한 시뮬레이션식이나 다른 변환기 종류도 정리해 드릴게요 😊

🔍 개념 정리

단락곡선(Short-Circuit Characteristic, SCC)은 동기발전기의 계자전류에 대한 단락 시 전기자전류의 변화를 나타낸 그래프입니다. 이 곡선은 일반적으로 거의 직선 형태를 띠는데, 그 이유는 다음과 같습니다:

🔧 동기발전기 단락 시 현상

  • 3상 단락 발생 시, 회로의 부하가 없어지고 출력단이 짧게 연결됨

  • 이때 흐르는 전기자전류의 위상은 계자전압보다 약 90도 뒤진 형태 → 순수한 유도성 전류

  • 이로 인해 철심이 포화되지 않고 선형적으로 자속이 증가 → 직선 특성 유지

🎯 전기자 반작용의 역할

  • 전기자 반작용(Armature Reaction): 전기자 전류가 생성한 자속이 계자 자속에 영향을 줌

  • 단락 시에는 전기자 반작용이 주자속을 감소시키는 방향으로 작용 (감자작용)

  • 철심의 포화가 해제 → 자기회로가 선형으로 동작함

  • 그 결과로 계자전류에 대한 단락전류가 직선적으로 증가 → 단락곡선이 직선 형태

  • 권선 수를 줄일 수 있고, 효율 및 역률이 높음

  • 그러나 절연성은 낮아 고압 차단기에 주의 필요

⚖️ 등가용량 vs 부하용량

  • 등가용량(Equiv. Capacity): 실제 변압기가 부담해야 하는 용량

  • 부하용량(Load Capacity): 최종적으로 부하에 공급되는 용량

단권변압기에서 등가용량은 전체 부하용량 중에서 자기적으로 연결된 부분을 제외한 유도적으로 전달되는 전력만을 기준으로 계산

✅ Y결선일 때 공식

등가용량부하용량=V1−V2V1\frac{\text{등가용량}}{\text{부하용량}} = \frac{V_1 - V_2}{V_1}

또는,

=1−V2V1= 1 - \frac{V_2}{V_1}

이 식은 변압기 동작 전압비 V2V1\frac{V_2}{V_1}에 따라 결정됩니다.

📌 결선 방식에 따른 비교

결선 방식 비율 계산 방식 특징
Y결선 위 식 그대로 → 정답은 2번 가장 일반적인 연결 방식
V결선 위 비율 × 23\frac{2}{\sqrt{3}} 접속코일이 2개만 사용됨
Δ결선 부하용량과 등가용량이 동일 → 비율은 3번 완전 삼상

궁금하시면 각 결선 방식의 응용 예나 실제 용량계산 사례도 함께 풀어드릴게요 😊⚙️

⚡ 문제 개요

문제: "변압기의 보호에 사용되지 않는 것은?"

선택지:

  1. 온도계전기

  2. 과전류계전기

  3. 임피던스계전기

  4. 비율차등계전기

✅ 정답: 3번 → 임피던스계전기 📊 정답률: 54%

🔒 변압기 보호용 계전기 종류

변압기를 보호하기 위한 계전기들은 내부 고장, 온도 상승, 기계적 이상 등을 감지해 동작합니다. 주요 보호용 계전기들은 다음과 같습니다:

보호 계전기 기능
온도계전기 권선 온도 상승 시 경고 또는 차단 동작
과전류계전기 과도한 부하전류 또는 단락전류 시 차단
비율차등계전기 내부 고장 검출 (1차/2차 전류 비교로 감지)
부흐홀츠 계전기 절연유가 있는 변압기에서 내부 고장 시 발생되는 가스 검출
충격압력계전기 급작스러운 압력 변화에 반응 (내부 파손 등)
유온계(유온계전기) 절연유 온도 상승에 따른 보호 동작

→ 위 계전기들은 변압기 자체의 보호를 위한 전용 장치들입니다.

❌ 임피던스계전기는 왜 정답인가?

  • 임피던스계전기(Impedance Relay)는 송전선로 보호용 계전기입니다.

  • 전류 및 전압의 비율로 계산된 임피던스 값에 따라 단락 위치를 판단하고 선로상의 고장 보호에 사용됩니다.

  • 거리계전기의 일종으로, 변압기 내부 고장에는 적합하지 않음.

→ 따라서 변압기 보호에 사용되지 않기 때문에 3번이 정답입니다.

60. E를 전압, r을 1차로 환산한 저항, χ를 1차로 환산한 리액터스라고 할 때 유도전동기의 원선도에서 원의 지름을 나타내는 것은?(정답률:57%)      
     1. E·r
     2. E·χ
     3. E/x
     4. E/r

     정답 : [3]

해설

다음은 문제 60번에 대한 상세 설명입니다:

🌀 문제 개요

문제: E를 전압, r을 1차로 환산한 저항, χ를 1차로 환산한 리액턴스라고 할 때, 유도전동기의 원선도(슬립원 또는 전력원선도)에서 원의 지름은 무엇인가?

선택지:

  1. E·r

  2. E·χ

  3. E/χ ✅

  4. E/r

✅ 정답: 3번 📊 정답률: 57%

🔍 원선도란?

유도전동기의 원선도(Circle Diagram)는 전동기의 전류, 전력, 토크 등의 운전 특성을 시각적으로 표현한 도식입니다. 전동기의 슬립(s)에 따른 변화나, 효율·전력분포를 이해하는 데 사용됩니다.

📘 핵심 개념

🎯 원의 지름의 의미:

  • 원선도의 지름(D)은 전동기 입력 전압 E와 리액턴스 χ에 의존합니다.

  • 전기적 모델에서 전압에 의해 흐르는 전류가 리액턴스에 의해 제한되기 때문에, 이 관계식은 다음과 같이 표현됩니다:

D∝EχD \propto \frac{E}{χ}

→ 따라서, 전압에 비례하고 리액턴스에 반비례하는 3번 E/χ가 정답!

❌ 틀린 선택지 해설

번호 식 틀린 이유
1번 E·r 저항은 손실 요소일 뿐, 원의 지름과 직접적인 관계 없음
2번 E·χ χ가 크면 지름이 작아지므로, 곱셈은 반대 개념
4번 E/r r은 전기손실 관련 요소이며, 지름 결정과는 무관함
