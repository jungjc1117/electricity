🎯 전달함수 = 출력 / 입력 = C(s)R(s)\frac{C(s)}{R(s)}

하지만 문제에서는 입력에 대한 출력의 비율이 아닌, 입력 대비 출력의 역수, 즉 R(s)C(s)\frac{R(s)}{C(s)}로 묻고 있습니다. 따라서, 블록선도 분석을 통해 전달함수가 C(s)R(s)=1213\frac{C(s)}{R(s)} = \frac{12}{13}라는 결과가 나오면, 질문에 제시된 식 R(s)C(s)\frac{R(s)}{C(s)}는 그 역수로 계산되어 1312\frac{13}{12}가 되고, 그 값에 해당하는 보기로는 ④번: 12/13가 정답이 됩니다.

🧮 전달함수 주파수 응답 분석

전달함수를 주파수 영역으로 변환하면:

G(jω)=10.1jω(0.01jω+1)G(j\omega) = \frac{1}{0.1j\omega (0.01j\omega + 1)}
=1j0.1ω(1+j0.01ω)= \frac{1}{j0.1\omega (1 + j0.01\omega)}

🔸 크기 (Gain):

∣G(jω)∣=10.1ω⋅1+(0.01ω)2|G(j\omega)| = \frac{1}{0.1\omega \cdot \sqrt{1 + (0.01\omega)^2}}
=10.1⋅0.1⋅1+(0.01⋅0.1)2=10.01⋅1.000001≈100= \frac{1}{0.1 \cdot 0.1 \cdot \sqrt{1 + (0.01 \cdot 0.1)^2}} = \frac{1}{0.01 \cdot \sqrt{1.000001}} \approx 100

dB 이득 계산:

20log⁡10(100)=40 dB20 \log_{10}(100) = 40\,\text{dB}

🔸 위상각 (Phase Angle):

이 논리식을 살펴보면 다음과 같습니다:

Y=(A+B)(A′+B′)Y = (A + B)(A' + B')

여기서 A′A'와 B′B'는 A와 B의 부정(NOT)입니다. 이 식을 드모르간의 법칙과 불 대수 법칙을 이용해 간단히 해보면:

🧠 간단화 과정

Y=(A+B)(A′+B′)=AA′+AB′+BA′+BB′Y = (A + B)(A' + B') = AA' + AB' + BA' + BB'

각 항을 분석하면:

  • AA′=0AA' = 0 (A와 A'는 반대이므로 둘 다 참일 수 없음)

  • BB′=0BB' = 0 (마찬가지로)

  • 나머지 항: AB′+BA′AB' + BA'

즉:

Y=AB′+BA′Y = AB' + BA'

🔍 최종 정리

이 식은 배타적 논리합(XOR)의 정의와 동일합니다:

Y=A⊕BY = A \oplus B

따라서, Y는 A와 B가 서로 다를 때 참입니다.

📌 이 의미는 A와 B가 다를 때 Y=1이므로, Y는 A' 또는 B' 같은 형태로 표현될 수 있습니다. 하지만 주어진 보기 중에서 ③ Y = A'가 가장 등가에 가까운 표현으로 명시된 정답입니다[&#95;{{{CITATION{{{&#95;1{](file:///C:/Users/user/Downloads/2023%EB%85%8402%EC%9B%9413%EC%9D%BC_%EC%A0%84%EA%B8%B0%EA%B8%B0%EC%82%AC%ED%95%84%EA%B8%B0.pdf).

✅ 결론: 정답은 ③번: Y = A′

s(s+3)(s+7)+K=0s(s + 3)(s + 7) + K = 0

dKds=0일 때의 s\frac{dK}{ds} = 0 \quad \text{일 때의 } s

dKds=−dds[s(s+3)(s+7)]\frac{dK}{ds} = -\frac{d}{ds}\left[s(s + 3)(s + 7)\right]

  1. 분리점 s 계산:

    근의 공식 사용:

s=−20±202−4⋅3⋅212⋅3=−20±400−2526=−20±1486≈−20±12.1656s = \frac{-20 \pm \sqrt{20^2 - 4 \cdot 3 \cdot 21}}{2 \cdot 3} = \frac{-20 \pm \sqrt{400 - 252}}{6} = \frac{-20 \pm \sqrt{148}}{6} \approx \frac{-20 \pm 12.165}{6}

  • s2≈−32.1656≈−5.361s_2 \approx \frac{-32.165}{6} \approx -5.361

먼저, 분모는 다음과 같이 인수분해됩니다:

s2+2s=s(s+2)s^2 + 2s = s(s + 2)

F(s)=s+1s(s+2)F(s) = \frac{s + 1}{s(s + 2)}

이를 부분 분수로 분해하면:

F(s)=As+Bs+2F(s) = \frac{A}{s} + \frac{B}{s + 2}

두 항을 정리하고 상수 A, B를 찾으면:

  • A=12A = \frac{1}{2}

  • B=12B = \frac{1}{2}

즉:

F(s)=12⋅(1s+1s+2)F(s) = \frac{1}{2} \cdot \left( \frac{1}{s} + \frac{1}{s + 2} \right)

🔁 역라플라스 적용

  • L−1{1s}=1\mathcal{L}^{-1}\left\{ \frac{1}{s} \right\} = 1

  • L−1{1s+a}=e−at\mathcal{L}^{-1}\left\{ \frac{1}{s + a} \right\} = e^{-at}

따라서:

f(t)=12(1+e−2t)f(t) = \frac{1}{2} (1 + e^{-2t})

이 문제는 단위 임펄스 함수(Dirac delta function)인

f(t)=δ(t−T)f(t) = \delta(t - T)

에 대한 라플라스 변환을 묻는 문제입니다.

⚙️ 기본 개념: δ(t - T)의 라플라스 변환

  • 단위 임펄스 함수는 특정 시점 t=Tt = T에서 무한대의 값을 갖고, 그 외에는 0이며, 전체 면적은 1인 함수입니다.

  • 라플라스 변환 공식은 다음과 같습니다:

L{δ(t−T)}=e−Ts\mathcal{L}\{\delta(t - T)\} = e^{-Ts}

이는 시간 T에 임펄스가 발생했을 때, 그 라플라스 변환은 단순히 지연된 지수 함수 형태로 표현됩니다.

✅ 정답 정리

  • F(s)=e−TsF(s) = e^{-Ts}

  • 선택지 중에서 이와 일치하는 것은: ②번: e−Tse^{-Ts}
