     1. 전파
     2. 혼촉
     3. 단락
     4. 정전기

     정답 : [1]

해설

📡 정답: 전파 — 왜 그럴까요?

주변 기출 콘텐츠와 문맥을 참고했을 때, 이 문제는 전기설비기술기준 관련 전기울타리나 통신 장치, 또는 고주파 간섭 등을 다루는 항목에서 출제된 것으로 보입니다. 특히 전력선 반송통신(PLC) 회로, 전기설비 주변에서 발생할 수 있는 영향 중 하나로 전파에 의한 간섭이 자주 언급되죠.

🔍 관련 개념 정리

  • 전파(電波): 전자기파의 일종으로 공기나 진공 중에 전자기 에너지를 전달하며, 통신에 사용되는 주된 매체입니다.

  • 혼촉: 서로 다른 전선이나 도체가 잘못 접촉되어 이상 전류가 흐르는 현상

  • 단락: 전기적 회로가 비정상적으로 연결돼 큰 전류가 흐르는 상태

  • 정전기: 마찰 등에 의해 도체에 축적되는 전기. 순간적인 방전 형태로 나타날 수 있음

📘 예상되는 문제 배경

  • 전파 방해에 대한 대책 혹은 전력선 통신에서 결합 장치 보호 회로의 설명일 가능성이 높습니다.

  • CC 커패시터(결합 커패시터), LF(결합 필터), L1~L3(방전갭과 피뢰기)가 있는 구성에서 발생할 수 있는 간섭 요인 중 하나로 전파가 정답이 되는 문맥

     2. 금속덕트 공사에 의하여 시설하는 경우
     3. 합성수지관 공사에 의하여 시설하는 경우
     4. 후강전선관 공사에 의하여 시설하는 경우

해설

📌 정답: 1번 – 버스덕트 공사에 의하여 시설하는 경우

🔍 문제 요점

옥내 저압 전선에 나전선을 사용할 수 있는 예외 조건을 묻는 문제입니다. 일반적으로 나전선(피복이 없는 전선)은 위험성을 고려해 실내에서 사용하는 것이 제한되지만, 특정 공법에서는 허용됩니다.

✅ 나전선 사용 가능한 공사 방식

  • 애자사용공사 : 전선을 애자 위에 설치해 외부와 직접 접촉을 피함

  • 버스덕트공사 : 절연된 금속 덕트 내부에 전선을 설치하는 방식. 구조상 안전하게 보호되어 나전선도 허용됨

  • 라이팅덕트공사 : 주로 조명기구 연결용. 금속 덕트 내부에 구성되어 나전선 사용 가능

  • 접촉전선공사 : 전차선, 트롤리선 등에서 사용되며 구조적으로 안전한 방식

위의 공사 방식은 모두 공사 구조상 외부 접촉 가능성을 낮추기 때문에 예외적으로 나전선 사용이 허용됩니다.

보기 설명 왜 오답인가
2. 금속덕트공사 배선 덕트 내부에 절연전선을 넣는 방식 절연전선이 기본으로 요구되며, 나전선 사용 불가
3. 합성수지관공사 전선을 합성수지관에 넣어 배관하는 방식 기계적 보호는 가능하나, 나전선은 불허
4. 후강전선관공사 강한 금속관에 전선을 넣어 배관하는 방식 구조적으로 안전하지만, 절연전선만 사용 가능

문제

84. 그림은 전력선 반송통신용 결합장치의 보안장치이다. 여기에서 CC는 어떤 커패시터인가?(정답률:69%)      
     1. 결합 커패시터
     2. 전력용 커패시터
     3. 정류용 커패시터
     4. 축전용 커패시터

     정답 : [1]

해설

🌀 정답: 1번 – 결합 커패시터

🔍 CC는 왜 "결합 커패시터"일까요?

CC는 전력선 반송통신(PLC) 시스템에서 전력선과 통신 장비 간 신호를 주고받기 위한 핵심 소자입니다. 일반적으로 고주파 통신 신호를 전력선에 결합하거나 분리할 때 사용하는 커패시터를 가리킵니다.

  • 결합 커패시터(CC)는 AC 전력 신호는 차단하고, 고주파 통신 신호는 통과시킬 수 있도록 설계되어 있어

  • PLC 장치 내에서 통신 신호의 전력선 주입 역할을 담당합니다.

🧪 CC와 다른 커패시터 비교

항목 정의 용도 이 문제와의 관계
결합 커패시터 (CC) 고주파만 통과시키는 커패시터 전력선에 통신신호를 주입하거나 추출 ✅ PLC 결합장치에서 사용됨
전력용 커패시터 전력 계통에서 역률 개선 또는 전압 조정에 사용 변전소, 산업 설비 등 ❌ 통신과는 무관
정류용 커패시터 전원회로에서 전압 안정화/필터링 SMPS, 전자기기 회로 ❌ 고주파 결합 목적과 다름
축전용 커패시터 에너지 저장 목적 커패시터뱅크, UPS 등 ❌ 결합장치에서는 사용되지 않음

📡 관련 부품 요약

이 보안장치 구성은 통신과 전력 보호를 위한 복합 구조입니다:

  • CC: 결합 커패시터 (통신 결합용)

  • CF: 결합 필터 (노이즈 제거)

  • DR: 배류 선륜 (전류 차단 보호)

  • L1~L3: 방전갭 또는 피뢰기 (전압 방호)

  • F: 포장퓨즈 (과전류 보호)

  • FD: 동축케이블 (통신 전송)

  • S, E: 접지 구성 및 개폐기 (안전장치)

🚧 정답: 3번 – 5m 이상

📌 문제 설명

이 문제는 교량의 윗면에 고압 전선로를 시설할 경우, 안전 확보를 위해 필요한 최소 높이 기준을 묻고 있습니다. 사람 또는 차량이 통행하는 장소인 교량은 접촉사고 위험이 있기 때문에 특별한 설치 기준이 적용됩니다.

✅ 시설 기준 요약

  • 교량 노면 상 전선의 설치 높이:

    • 저압 및 고압 모두: 5m 이상

    • 이는 「전기설비기술기준」에서 규정된 최소 설치 높이이며, 외부 접촉 가능성을 최소화하고, 차량 운행 및 작업자의 안전 확보를 위한 기준입니다.

보기 설명 왜 오답인가?
1. 3m 전기기계기구 내 또는 벽체 내부 배선에는 해당 가능 교량은 노면 상 설치이므로 불충분
2. 4m 통행 가능 장소에서는 부족 고압은 5m 이상이 반드시 요구됨
4. 6m 지나치게 과도함 기준보다 높음 → 옳은 답이지만 "틀린 것은"이 아니라 "맞는 것은"을 묻는 문제에선 불필요하게 높음

해설

🛠️ 정답: 2번 – 발전기의 회전수 및 주파수

🔍 문제 요약

이 문제는 발전소에 계측 장치를 설치해야 하는 대상을 묻고 있으며, 즉 필수적으로 측정해야 하는 요소인지 아닌지를 판단해야 하는 문제입니다.

✅ 어떤 항목들이 계측 대상인가?

발전소에서는 안전성과 효율적인 운전 감시를 위해 다음 항목들에는 반드시 계측 장치를 설치해야 합니다:

  • 특고압용 변압기 온도 → 내부 고열로 절연 열화 가능성 있음 → 계측 필요

  • 발전기 베어링 온도 (수중 메탈 제외) → 마찰 열로 인한 이상 방지 목적 → 계측 필요

  • 발전기 고정자 온도 → 절연 파괴나 열화 예방 → 계측 필요

❌ 왜 2번은 계측 대상이 아닌가?

  • 발전기의 회전수 및 주파수는 계측이 필수 항목은 아닙니다.

    • 주파수는 계통 전체의 운전 상태에 따라 자동 제어되고 감시되며

    • 회전수는 간접적으로 주파수와 연계되어 있고

    • 별도의 계측 장치를 필수적으로 설치하진 않음

📌 참고: VIP 항목

발전소의 주요 계측 대상은 다음과 같이 분류됩니다:

구분 계측 대상 예시
발전기 전압, 전류, 전력, 온도 등
변압기 온도, 절연 저항
설비보호 베어링, 절연파괴 방지 기준

98. 최대사용전압이 7 ㎸를 초과하는 회전기의 절연내력 시험은 최대사용전압의 몇 배의 전압 (10,500 V 미만으로 되는 경우에는 10,500 V)에서 10분간 견디어야 하는가?(정답률:69%)      
     1. 0.92
     2. 1
     3. 1.1
     4. 1.25

     정답 : [4]

해설

✅ 정답: 4번 – 1.25배

📌 문제 해설 요약

이 문제는 회전기(발전기, 전동기 등)의 절연내력 시험 기준을 묻는 것으로, 최대사용전압이 7kV 초과일 때 적용되는 시험 전압 조건을 알고 있어야 정확하게 답할 수 있습니다.

⚙️ 회전기의 절연내력 시험 기준

최대사용전압 기준 시험 전압 배수 최저 시험전압 기준
7kV 이하 1.5배 500V 이상
7kV 초과 1.25배 10,500V 이상

즉, 7kV 초과 회전기는 최대사용전압 × 1.25의 전압에서 10분간 절연내력 시험을 견뎌야 하며, 계산된 값이 10,500V보다 작을 경우 10,500V로 시험을 진행해야 한다는 것이 핵심입니다.

🧪 예시 계산

  • 최대사용전압: 6.6kV → 7kV 이하 → 1.5배 → 시험전압 = 6.6 × 1.5 = 9.9kV → ✅ 500V 이상이면 통과

  • 최대사용전압: 8kV → 7kV 초과 → 1.25배 → 시험전압 = 8 × 1.25 = 10kV → ⛔ 10,500V 미만 → 10,500V로 시험 진행
